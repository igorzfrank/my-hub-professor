1 🔄 O QUE SÃO LAÇOS DE REPETIÇÃO?
🧠 Repetir sem copiar e colar
Laços de repetição (loops) permitem executar um bloco de código várias vezes sem precisar escrevê-lo repetidamente. Em vez de fazer console.log 100 vezes, você cria um loop que repete a ação 100 vezes. É assim que programas processam listas, geram relatórios e automatizam tarefas.
📐Comparação: sem loop vs com loop
❌ Sem loop (repetitivo)
console.log(1);
console.log(2);
console.log(3);
console.log(4);
console.log(5);
// E se fossem 1000?
✅ Com loop (elegante)
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
// Funciona para 5 ou 5000!
🧠 Todo loop tem 3 partes essenciais
1. Inicialização: De onde começa (ex: let i = 0)
2. Condição de parada: Até quando continua (ex: i < 10)
3. Incremento: Como avança (ex: i++)

Se a condição de parada nunca for atingida, o loop vira um loop infinito e trava o programa.
2 🔧 for E while — Os Dois Principais
1for — Quando você sabe quantas vezes repetir
O for é o loop mais usado. Ele concentra inicialização, condição e incremento em uma única linha. Ideal para percorrer arrays (você sabe o tamanho) ou repetir um número fixo de vezes.
// Estrutura do for
for (inicialização; condição; incremento) {
  // código a repetir
}

// Exemplo: contando de 1 a 5
for (let i = 1; i <= 5; i++) {
  console.log(`Volta número ${i}`);
}

2while — Quando você não sabe quantas vezes repetir
O while repete enquanto a condição for verdadeira. É ideal quando você não sabe antecipadamente quantas repetições serão necessárias — como esperar o usuário digitar uma senha correta.
// Estrutura do while
while (condição) {
  // código a repetir
}

// Exemplo: contagem regressiva
let contador = 5;
while (contador > 0) {
  console.log(contador);
  contador--; // sem isso = loop infinito!
}
⚠️ Cuidado: Se a condição do while nunca se tornar false, o loop nunca termina. Sempre garanta que algo dentro do loop altere a condição de parada.
3 🔀 do...while, break E continue
1do...while — Execute pelo menos uma vez
A diferença para o while: o do...while executa o bloco antes de verificar a condição. Isso garante que o código rode pelo menos uma vez.
let x = 10;
do {
  console.log(x);
  x++;
} while (x < 5);
Mesmo com condição false,
executa uma vez (exibe 10).
// Exemplo prático
let senha;
do {
  senha = prompt('Senha:');
} while (senha !== '123');
Pede a senha ao menos uma vez,
depois repete se estiver errada.
2break e continue — Controle total do loop
Comandos especiais para controlar o fluxo dentro de qualquer loop.
🛑 break
Interrompe o loop imediatamente e sai dele, ignorando qualquer iteração restante.
for (let i = 1; i <= 10; i++) {
  if (i === 5) break;
  console.log(i);
}
// Exibe: 1, 2, 3, 4
⏭️ continue
Pula para a próxima iteração, ignorando o resto do bloco atual, mas sem sair do loop.
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue;
  console.log(i);
}
// Exibe: 1, 2, 4, 5
4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Pratique no console ou em um arquivo .js
Exercício 1 — Contagem com for
Use um for para exibir no console todos os números de 1 a 10. Depois faça o mesmo exibindo apenas os números pares (dica: use if com %).
Exercício 2 — Tabuada
Crie uma variável numero = 7. Use um for para exibir a tabuada completa desse número: "7 x 1 = 7", "7 x 2 = 14"... até "7 x 10 = 70".
Exercício 3 — Contagem regressiva com while
Use um while para fazer uma contagem regressiva de 10 até 1. Ao final, exiba "Feliz Ano Novo!". Cuidado para não criar um loop infinito.
Exercício 4 — break e continue
Crie um for de 1 a 20. Use continue para pular os números múltiplos de 3. Use break para parar o loop quando chegar ao número 16.
💡 Desafio extra: Crie um array numeros = [12, 5, 8, 21, 3, 14, 7]. Use um for para percorrer o array e encontrar: (1) o maior número, (2) o menor número e (3) a soma de todos os valores. Exiba os três resultados no console. Dica: crie variáveis antes do loop para armazenar esses valores.
