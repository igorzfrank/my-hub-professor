1 📚 O QUE É UM ARRAY?
🧠 Definição
Um Array é uma lista ordenada de valores. Pense em um array como um armário com gavetas numeradas: cada gaveta (índice) guarda um item, e a numeração começa do 0 (zero). Arrays podem armazenar qualquer tipo de dado — strings, numbers, booleans e até outros arrays e objetos.

frutas
Índice 0
🍎 maçã

// Criando um array
let frutas = ['maçã', 'banana', 'laranja', 'uva'];

// Acessando por índice
console.log(frutas[0]);  // 'maçã'
console.log(frutas[2]);  // 'laranja'

// Último elemento (independente do tamanho)
console.log(frutas[frutas.length - 1]); // 'uva'
💡 Arrays podem conter tipos mistos
Diferente de outras linguagens, JavaScript permite que um mesmo array tenha strings, numbers e booleans: let misto = ['texto', 42, true, 'outro']; — embora o mais comum seja manter o mesmo tipo.
2 🔧 MÉTODOS ESSENCIAIS — Manipulando Arrays
🧠 Métodos são funções que pertencem ao array
Arrays possuem métodos embutidos que permitem adicionar, remover, buscar e transformar itens. Você aciona um método usando .nomeDoMetodo() após o array. Abaixo, os métodos que você mais usará no dia a dia.
➕ push(item)
Adiciona um ou mais itens no final do array e retorna o novo tamanho.
frutas.push('morango');
➖ pop()
Remove o último item do array e retorna o item removido.
let ultimo = frutas.pop();
➕ unshift(item)
Adiciona um ou mais itens no início do array e retorna o novo tamanho.
frutas.unshift('melão');
➖ shift()
Remove o primeiro item do array e retorna o item removido.
let primeiro = frutas.shift();
🔍 indexOf(item)
Retorna o índice do item ou -1 se não encontrado.
✅ includes(item)
Retorna true se o item existe no array, false se não.
🔀 reverse()
Inverte a ordem dos itens no array.
let numeros = [10, 20, 30];

numeros.push(40);       // [10, 20, 30, 40]
numeros.pop();          // [10, 20, 30]
numeros.unshift(5);     // [5, 10, 20, 30]
numeros.shift();        // [10, 20, 30]

console.log(numeros.indexOf(20));  // 1
console.log(numeros.includes(50)); // false
💡 Regra prática: push e pop são os mais usados no dia a dia. unshift e shift são menos performáticos (reorganizam todos os índices). Para listas grandes, prefira push/pop.
3 🔄 PERCORRENDO ARRAYS — Um item por vez
🧠 Por que percorrer um array?
Na maioria das vezes, você não quer apenas guardar itens — quer fazer algo com cada um deles. Exibir todos os nomes de uma lista, calcular o total de uma compra, filtrar itens... Para isso, você precisa percorrer (iterar sobre) o array. O método mais direto e moderno é o forEach.
1forEach — O método mais direto
Executa uma função para cada item do array. Recebe uma função callback que pode acessar o item atual, o índice e o array completo.
let nomes = ['Ana', 'Bruno', 'Carla'];

nomes.forEach(function(item, indice) {
  console.log(`${indice}: ${item}`);
});

// Resultado:
// 0: Ana
// 1: Bruno
// 2: Carla
2for clássico — Mais controle
O laço for tradicional oferece mais controle sobre a iteração. Você define o início, a condição de parada e o incremento manualmente. Muito usado quando se precisa do índice para manipular outros arrays.
let frutas = ['maçã', 'banana', 'uva'];

for (let i = 0; i < frutas.length; i++) {
  console.log(`Fruta ${i + 1}: ${frutas[i]}`);
}
💡 Qual usar? Para a maioria dos casos, forEach é mais legível e rápido de escrever. Use o for clássico quando precisar de mais controle sobre o índice ou quando quiser interromper o laço antes do fim (com break, que veremos em aulas futuras).
4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Pratique no console ou em um arquivo .js
Exercício 1 — Criando e acessando
Crie um array com 5 nomes de cidades brasileiras. Exiba no console o primeiro, o terceiro e o último elemento da lista.
Exercício 2 — Manipulando a lista
Usando o array do exercício 1: adicione uma cidade no final (push), remova a primeira (shift) e exiba o array resultante com seu novo tamanho (length).
Exercício 3 — Verificando existência
Crie um array de números. Verifique se o número 7 existe no array usando includes(). Depois descubra em qual posição ele está usando indexOf().
Exercício 4 — Percorrendo com forEach
Crie um array com suas 3 comidas favoritas. Use forEach para exibir cada comida numerada: "1. Pizza", "2. Sushi", etc.
💡 Desafio extra: Crie um array precos = [29.90, 15.50, 45.00, 9.99]. Use forEach para calcular o valor total da compra. Exiba: "Total: R$ [valor]". Depois adicione mais um preço com push e recalcule. Dica: crie uma variável let total = 0 antes do forEach e vá somando dentro dele.
