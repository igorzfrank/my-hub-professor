Arrays — Listas, Índices e Métodos
15 exercícios · Criação · Índices · push/pop · length · forEach · includes · indexOf
Array Métodos forEach Prática
1-5 📚 CRIAÇÃO E ACESSO A ARRAYS
1
Criando seu primeiro array
Crie um array chamado cores com 5 nomes de cores. Exiba o array completo no console e também exiba o valor de cores.length para verificar o tamanho.
2
Acessando por índice
Usando o array cores do exercício 1, exiba no console: o primeiro elemento (índice 0), o terceiro elemento (índice 2) e o último elemento (use cores.length - 1).
3
Array de números
Crie um array chamado idades com 6 valores numéricos diferentes. Exiba o array, seu tamanho (length) e a soma do primeiro com o último elemento (use os índices).
4
Array misto
Crie um array chamado misturado contendo: uma string, um number, um boolean, outro string e outro number. Use typeof para verificar o tipo do terceiro elemento do array.
5
Array vazio e preenchimento
Crie um array vazio let lista = []. Depois, usando os índices manualmente, atribua valores nas posições 0, 1 e 2. Exiba o array e seu length. O que acontece se você atribuir diretamente na posição 5?
6-10 🔧 MÉTODOS DE MANIPULAÇÃO
6
push e pop na prática
Crie um array let animais = ['gato', 'cachorro']. Use push para adicionar 'pássaro' e 'peixe'. Depois use pop para remover o último. Exiba o array e o elemento removido em cada etapa.
7
unshift e shift na prática
Usando o mesmo array do exercício 6, use unshift para adicionar 'leão' no início. Depois use shift para remover o primeiro elemento. Compare o comportamento com push/pop.
8
Procurando com indexOf
Crie um array let frutas = ['maçã', 'banana', 'laranja', 'uva', 'banana']. Use indexOf('banana') para encontrar a posição da primeira 'banana'. Depois teste indexOf com uma fruta que não existe no array.
9
Verificando com includes
Crie um array let numeros = [3, 7, 12, 9, 5, 8]. Use includes() para verificar se o número 12 existe no array, se o número 15 existe e se o número 7 existe. Exiba true ou false para cada verificação.
10
Invertendo com reverse
Crie um array let letras = ['A', 'B', 'C', 'D', 'E']. Use reverse() para inverter a ordem. Exiba o array antes e depois. Atenção: reverse modifica o array original!
11-15 🔄 PERCORRENDO COM forEach E for
11
Primeiro forEach
Crie um array paises = ['Brasil', 'Argentina', 'Chile', 'Peru', 'Uruguai']. Use forEach para exibir cada país no console com a frase: "País: [nome]".
12
forEach com índice
Usando o array de países, use forEach com o parâmetro de índice para exibir: "1º - Brasil", "2º - Argentina", "3º - Chile"... (lembre-se de somar 1 ao índice).
13
Percorrendo com for clássico
Crie um array numeros = [10, 20, 30, 40, 50]. Use um for clássico para percorrer o array e exibir cada número multiplicado por 2. O resultado deve ser: 20, 40, 60, 80, 100.
14
Somando valores com forEach
Crie um array valores = [15, 22, 8, 34, 11]. Use forEach para calcular a soma de todos os valores. Exiba: "A soma total é [resultado]". Dica: crie let soma = 0 antes do forEach.
15
Mini projeto: Lista de compras
Crie um script completo com: (1) array vazio let compras = [], (2) use push para adicionar 5 itens de supermercado, (3) use forEach para exibir todos os itens numerados, (4) verifique com includes se 'arroz' está na lista, (5) remova o último item com pop e exiba o item removido, (6) use forEach novamente para mostrar a lista atualizada com o total de itens (length).
💡 DICAS PARA RESOLUÇÃO
✅ O que você deve praticar
📚 Criar arrays com colchetes [ ]
🔢 Acessar elementos por índice (começa em 0)
📏 Usar .length para saber o tamanho
➕ Adicionar com push() e unshift()
➖ Remover com pop() e shift()
🔍 Buscar com indexOf() e includes()
🔀 Inverter com reverse()
🔄 Percorrer com forEach() e for
⚠️ Erros comuns para evitar
❌ Esquecer que índices começam em 0 (não em 1)
❌ Usar length como índice (o último é length - 1)
❌ Confundir push/pop (final) com unshift/shift (início)
❌ Achar que indexOf retorna true/false (retorna índice ou -1)
❌ Esquecer que reverse() modifica o array original
❌ Usar forEach esperando retornar um valor (ele não retorna nada)
💡 Dica de ouro: Arrays são a estrutura de dados mais usada em JavaScript. Dominar a criação, manipulação e iteração de arrays é essencial para tudo que virá depois: objetos, funções e manipulação do DOM. Pratique bastante! 🚀
