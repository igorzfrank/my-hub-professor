1 🗂️ O QUE É UM OBJETO?
🧠 Definição
Um Objeto é uma coleção de pares chave-valor. Enquanto arrays usam índices numéricos (0, 1, 2...), objetos usam nomes descritivos (propriedades) para acessar valores. Pense em um objeto como uma ficha cadastral: cada campo tem um rótulo (nome, idade, email) e um valor associado. São criados com { }.

Objeto pessoa com 4 propriedades. Cada propriedade tem uma chave (rótulo) e um valor.
// Criando um objeto
let pessoa = {
  nome: 'Carlos',
  idade: 28,
  cidade: 'Curitiba',
  logado: true
};

// Acessando propriedades (duas formas)
console.log(pessoa.nome);      // 'Carlos' (notação de ponto)
console.log(pessoa['idade']);  // 28 (notação de colchetes)
💡 Ponto vs Colchetes: quando usar cada um?
Use notação de ponto (objeto.propriedade) sempre que souber o nome da propriedade. Use notação de colchetes (objeto['propriedade']) quando o nome da propriedade estiver em uma variável ou tiver caracteres especiais.
2 🔧 MANIPULANDO OBJETOS
🧠 Adicionar, alterar e remover propriedades
Diferente de muitas linguagens, objetos JavaScript são dinâmicos: você pode adicionar, alterar e remover propriedades a qualquer momento, mesmo depois de criados.
➕ Adicionar
Basta atribuir um valor a uma nova chave.
pessoa.email = 'c@email.com';
✏️ Alterar
Atribua um novo valor a uma chave existente.
pessoa.idade = 29;
🗑️ Remover
Use o operador delete.
delete pessoa.cidade;
let carro = {
  marca: 'Toyota',
  modelo: 'Corolla',
  ano: 2022
};

// Adicionar
carro.cor = 'prata';

// Alterar
carro.ano = 2023;

// Remover
delete carro.modelo;

console.log(carro); // {marca: 'Toyota', ano: 2023, cor: 'prata'}
💡 Verificando se uma propriedade existe: Use 'propriedade' in objeto — retorna true ou false. Ex: 'email' in pessoa. Isso evita erros ao acessar propriedades inexistentes.
3 🔄 PERCORRENDO OBJETOS E ARRAYS DE OBJETOS
1for...in — Percorrendo propriedades
O laço for...in percorre todas as chaves de um objeto, permitindo acessar cada propriedade dinamicamente.
let pessoa = { nome: 'Ana', idade: 25, cidade: 'SP' };

for (let chave in pessoa) {
  console.log(`${chave}: ${pessoa[chave]}`);
}

// Resultado:
// nome: Ana
// idade: 25
// cidade: SP
2Array de objetos — O padrão mais comum
No mundo real, você raramente trabalha com um único objeto. O mais comum são listas de objetos (como produtos, alunos, pedidos). Aqui você combina tudo que aprendeu sobre arrays e objetos.
let alunos = [
  { nome: 'João', nota: 8.5 },
  { nome: 'Maria', nota: 9.0 },
  { nome: 'Pedro', nota: 7.0 }
];

// Percorrendo com forEach
alunos.forEach(function(aluno) {
  console.log(`${aluno.nome}: ${aluno.nota}`);
});
3Métodos úteis: Object.keys() e Object.values()
Transformam um objeto em array, extraindo só as chaves ou só os valores. Muito útil para manipulação.
Object.keys(pessoa)
// ['nome','idade','cidade']
Object.values(pessoa)
// ['Ana', 25, 'SP']
4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Pratique no console ou em um arquivo .js
Exercício 1 — Criando seu primeiro objeto
Crie um objeto livro com as propriedades: titulo, autor, ano e paginas. Exiba no console uma frase: "O livro [titulo] foi escrito por [autor] em [ano]."
Exercício 2 — Manipulando propriedades
Usando o objeto do exercício 1: adicione uma propriedade editora, altere o ano e remova paginas. Exiba o objeto resultante.
Exercício 3 — Percorrendo com for...in
Crie um objeto celular com 4 propriedades (marca, modelo, ano, cor). Use for...in para exibir todas as chaves e valores no console.
Exercício 4 — Array de objetos
Crie um array com 3 objetos, cada um representando um filme (titulo, ano, genero). Use forEach para exibir: "[titulo] - [ano] - [genero]".
💡 Desafio extra: Crie um array com 5 objetos representando produtos de uma loja (nome, preco, categoria). Use forEach para: (1) exibir todos os produtos com preço, (2) calcular o valor total do estoque (soma de todos os preços) e (3) contar quantos produtos pertencem a uma categoria específica.
