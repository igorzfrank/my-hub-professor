1 🎯 O QUE VAMOS CONSTRUIR
🧠 Um CRUD completo de tarefas
Neste projeto, você vai construir uma Lista de Tarefas completa no navegador. O projeto integra todos os conceitos das aulas anteriores: variáveis, arrays, objetos, funções, condicionais, laços, manipulação do DOM e eventos. Você implementará as quatro operações fundamentais de um sistema: Criar, Ler, Atualizar e Deletar — o famoso CRUD.
📐Funcionalidades da Lista de Tarefas
➕
Adicionar tarefa
Digitar no campo e clicar em "Adicionar" (ou pressionar Enter).
📋
Listar tarefas
Exibir todas as tarefas em tela, com atualização automática.
✅
Marcar como concluída
Clicar na tarefa para riscá-la (alternar concluído/não concluído).
🗑️
Remover tarefa
Botão para excluir uma tarefa específica da lista.
🧠 Arquitetura do projeto
O coração do projeto é um array de objetos armazenando as tarefas. Cada tarefa tem: id (número único), texto (descrição) e concluida (boolean). Uma função renderizar() redesenha a lista na tela sempre que o array é modificado.

3 🧠 LÓGICA JAVASCRIPT PASSO A PASSO
🧠 Implemente na ordem abaixo
Cada etapa constrói sobre a anterior. Teste após cada passo para garantir que tudo funciona antes de seguir adiante.
1Array de tarefas e função renderizar
Crie o array global e a função que redesenha a lista na tela sempre que o array muda. Esta é a base de tudo.
let tarefas = [];
let contadorId = 0;

function renderizar() {
  let lista = document.querySelector('#lista');
  lista.innerHTML = ''; // Limpa a lista

  tarefas.forEach(tarefa => {
    let item = document.createElement('li');
    item.textContent = tarefa.texto;
    lista.appendChild(item);
  });
}
2Adicionar tarefa (CREATE)
Capture o valor do input, valide se não está vazio, crie o objeto tarefa e adicione ao array. Depois chame renderizar().
function adicionarTarefa() {
  let input = document.querySelector('#entrada');
  let texto = input.value.trim();

  if (texto !== '') {
    contadorId++;
    tarefas.push({ id: contadorId, texto: texto, concluida: false });
    input.value = ''; // Limpa o campo
    renderizar();
  }
}
3Conectar o evento de clique
Vincule o botão e a tecla Enter à função de adicionar tarefa.
let btn = document.querySelector('#btnAdicionar');
btn.addEventListener('click', adicionarTarefa);

let input = document.querySelector('#entrada');
input.addEventListener('keydown', function(event) {
  if (event.key === 'Enter') adicionarTarefa();
});
4Melhorar a renderização (UPDATE e DELETE)
Adicione botão de excluir e evento de clique para marcar como concluída em cada item renderizado.
function renderizar() {
  let lista = document.querySelector('#lista');
  lista.innerHTML = '';

  tarefas.forEach(tarefa => {
    let item = document.createElement('li');
    item.textContent = tarefa.texto;
    if (tarefa.concluida) item.classList.add('concluida');

    // Clique para marcar/desmarcar
    item.addEventListener('click', () => {
      tarefa.concluida = !tarefa.concluida;
      renderizar();
    });

    // Botão de excluir
    let btnExcluir = document.createElement('button');
    btnExcluir.textContent = '❌';
    btnExcluir.addEventListener('click', (e) => {
      e.stopPropagation(); // Não marca ao excluir
      tarefas = tarefas.filter(t => t.id !== tarefa.id);
      renderizar();
    });

    item.appendChild(btnExcluir);
    lista.appendChild(item);
  });
}
4 🚀 DESAFIOS EXTRAS — MELHORE SEU PROJETO
💡 Depois que o projeto básico funcionar, tente implementar:
🔢 Contador de tarefas
Exiba "X tarefas (Y concluídas)" no topo ou rodapé da lista. Use filter para contar as concluídas.

Adicione botões "Todas", "Pendentes" e "Concluídas" que filtram a exibição. Use filter antes de renderizar.

✏️ Editar tarefa
Permita clicar duas vezes em uma tarefa para editar seu texto. Use prompt() ou um input que aparece no lugar do texto.
