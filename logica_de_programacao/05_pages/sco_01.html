1 📌 O QUE É JAVASCRIPT?
🧠 Definição
JavaScript é uma linguagem de programação interpretada que adiciona interatividade e dinamismo às páginas web. Criada em 1995 por Brendan Eich, hoje é uma das três tecnologias fundamentais da web junto com HTML (estrutura) e CSS (estilo). Com JavaScript você pode responder a cliques, validar formulários, criar animações, carregar dados de servidores e muito mais.

💡 Curiosidade
JavaScript foi criado em apenas 10 dias por Brendan Eich na Netscape. O nome foi uma jogada de marketing para pegar carona na fama do Java, mas as duas linguagens não têm relação técnica — são tão parecidas quanto "carro" e "carpete".
2 ⚙️ COMO O JAVASCRIPT FUNCIONA?
🧠 O Motor JavaScript
Todo navegador tem um motor JavaScript embutido que lê e executa o código. O mais famoso é o V8 (Google Chrome e Node.js). O processo é: o motor lê o código → compila em tempo real (JIT) → executa. Isso permite que o JavaScript rode em qualquer navegador moderno sem precisar de instalação adicional.
📍Onde colocar o código JavaScript
📄 No próprio arquivo HTML
Usando a tag <script> dentro do HTML. Pode ser no <head> ou antes do fechamento do </body>.
<script>
  alert('Olá, mundo!');
</script>
📁 Arquivo externo (.js)
Código em um arquivo separado com extensão .js, vinculado ao HTML com o atributo src.
<script src="script.js">
</script>
💡 Boa prática: Coloque seus scripts antes do fechamento do </body> ou use o atributo defer. Isso garante que o HTML carregue antes do JavaScript, evitando erros ao tentar manipular elementos que ainda não existem na página.
3 💻 PRIMEIROS COMANDOS
1console.log() — Sua melhor amiga
O comando console.log() exibe mensagens no console do navegador (F12). É a ferramenta mais usada para depurar e testar código.
console.log('Olá, JavaScript!');
console.log(42);
console.log(10 + 5);
2Variáveis — Guardando informações
Variáveis são como caixas etiquetadas onde guardamos dados para usar depois. Em JavaScript moderno, usamos let (pode mudar) e const (não pode mudar). Evite var (forma antiga).
// Variável que pode ser alterada
let nome = 'João';
let idade = 25;

// Constante — não pode ser alterada
const pi = 3.1415;
const pais = 'Brasil';
3Comentários — Explicando o código
Comentários são trechos ignorados pelo motor JavaScript. Servem para explicar o código para humanos (incluindo você no futuro).
// Comentário de uma linha
/* Comentário
de múltiplas
linhas */
4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Pratique no console do navegador (F12)
Exercício 1 — Primeiro log
Abra o console do navegador (F12) e digite: console.log("Meu nome é [seu nome]")
Exercício 2 — Criando variáveis
Crie uma variável let nome = "seu nome" e uma constante const ano = 2026. Depois use console.log(nome, ano) para exibir as duas.
Exercício 3 — Comentários
Crie um arquivo HTML com uma tag <script>. Dentro dele, escreva um comentário explicando o que o script faz e um console.log com uma mensagem.
Exercício 4 — Mini desafio
Crie variáveis para armazenar: seu nome, sua idade e sua cidade. Depois use console.log para exibir uma frase completa como: "Meu nome é João, tenho 25 anos e moro em São Paulo."
💡 Desafio extra: Descubra o que acontece se você tentar reatribuir um valor a uma const. Faça o teste: const x = 10; x = 20; console.log(x); — Qual erro aparece? Isso te ajudará a entender quando usar let e quando usar const.
