1 🧩 O QUE SÃO FUNÇÕES?
🧠 Um bloco de código com nome
Uma função é um bloco de código reutilizável que executa uma tarefa específica. Pense em uma função como uma máquina: você coloca algo (entrada/parâmetros), ela processa e devolve algo (saída/retorno) — ou simplesmente executa uma ação. Funções evitam repetição, organizam o código e facilitam a manutenção.

Toda função segue este fluxo: recebe dados, processa e devolve um resultado (ou executa uma ação).
❌ Sem função (repetitivo)
console.log('Olá, Ana!');
console.log('Olá, Bruno!');
console.log('Olá, Carla!');
// Código duplicado
✅ Com função (limpo)
function saudar(nome) {
  console.log(`Olá, ${nome}!`);
}
saudar('Ana');
saudar('Bruno');
saudar('Carla');
2 📥📤 PARÂMETROS E RETURN
🧠 Dados que entram e saem da função
Parâmetros são variáveis que recebem valores quando a função é chamada. return é a palavra que devolve um resultado para quem chamou a função. Uma função pode ter zero, um ou vários parâmetros — e pode ou não ter return.
1Parâmetros — A entrada da função
São declarados entre parênteses e recebem os valores (argumentos) passados na chamada da função.
// Um parâmetro
function dobro(num) {
  console.log(num * 2);
}
dobro(5); // 10
// Vários parâmetros
function soma(a, b) {
  console.log(a + b);
}
soma(10, 20); // 30
2return — A saída da função
Devolve um valor para quem chamou a função. Tudo abaixo de return é ignorado. Se não houver return, a função retorna undefined.
function calcularArea(largura, altura) {
  return largura * altura;
  // Nada aqui é executado
}

let area = calcularArea(5, 3);
console.log(area); // 15
💡 console.log vs return: console.log só exibe no console. return entrega o valor de volta para ser usado em outras partes do código (armazenar em variável, passar para outra função, etc.).
3 🏹 ARROW FUNCTIONS E ESCOPO
1Arrow Function — Sintaxe moderna
Forma mais curta de escrever funções usando => (flecha). Ideal para funções simples. Muito usada com métodos de array como forEach, map e filter.
// Função tradicional
function soma(a, b) {
  return a + b;
}

// Arrow function equivalente
const soma = (a, b) => {
  return a + b;
};

// Ainda mais curta (return implícito)
const soma = (a, b) => a + b;
2Escopo — Onde as variáveis são visíveis
Variáveis declaradas dentro de uma função só existem lá dentro (escopo local). Variáveis declaradas fora podem ser acessadas de qualquer lugar (escopo global).
✅ Global (acessível em todo lugar)
let nome = 'João';

function exibir() {
  console.log(nome);
}
exibir(); // 'João'
❌ Local (só dentro da função)
function criar() {
  let segredo = 'xyz';
}
console.log(segredo);
// ❌ Erro!
4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Pratique no console ou em um arquivo .js
Exercício 1 — Primeira função
Crie uma função chamada saudacao que recebe um nome como parâmetro e exibe no console: "Bem-vindo(a), [nome]!". Chame a função com 3 nomes diferentes.
Exercício 2 — Função com return
Crie uma função calcularMedia que recebe 3 notas e retorna a média. Armazene o resultado em uma variável e exiba: "A média é [valor]".
Exercício 3 — Arrow function
Converta a função do exercício 2 para uma arrow function. Use a forma mais curta possível (return implícito). Teste com os mesmos valores.
Exercício 4 — Função com array
Crie uma função listarNomes que recebe um array de nomes e usa forEach para exibir cada nome numerado no console.
💡 Desafio extra: Crie uma função verificarIdade que recebe uma idade e retorna uma string: "Criança", "Adolescente", "Adulto" ou "Idoso". Use if/else if. Depois crie um array com 5 idades diferentes e use forEach para chamar a função para cada idade e exibir o resultado. Dica: combine função + array + forEach + condicionais em um só exercício!
