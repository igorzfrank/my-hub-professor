1 🌳 O QUE É O DOM?
🧠 A ponte entre HTML e JavaScript
O DOM (Document Object Model) é a representação do HTML como uma árvore de objetos que o JavaScript pode acessar e modificar. É através do DOM que seu código "conversa" com a página: lê textos, altera estilos, responde a cliques, adiciona ou remove elementos. Toda interação dinâmica em uma página web passa pelo DOM.

O JavaScript acessa qualquer elemento da árvore e pode ler, alterar, adicionar ou remover.
🧠 O objeto document
Toda manipulação do DOM começa pelo objeto document. Ele representa a página inteira e fornece métodos para encontrar elementos: document.querySelector(), document.getElementById() e outros. É a porta de entrada do JavaScript para o HTML.
2 🔍 SELECIONANDO E MODIFICANDO ELEMENTOS
1Encontrando elementos na página
Use seletores CSS para encontrar elementos. O método mais versátil é o querySelector (retorna o primeiro) e querySelectorAll (retorna todos em uma NodeList).
// Por tag, classe ou id
let titulo = document.querySelector('h1');
let botao = document.querySelector('.btn');
let header = document.querySelector('#cabecalho');
// Múltiplos elementos
let paragrafos = document.querySelectorAll('p');

// querySelectorAll retorna NodeList
paragrafos.forEach(p => {
  console.log(p.textContent);
});
2Modificando conteúdo e estilo
Depois de selecionar um elemento, você pode ler ou alterar seu conteúdo, atributos e estilo diretamente.
// Conteúdo de texto
elemento.textContent = 'Novo texto';
// HTML interno
elemento.innerHTML = '<strong>Oi</strong>';
// Estilo CSS
elemento.style.color = 'red';
3 🖱️ EVENTOS, CLASSES E CRIAR ELEMENTOS
1Eventos — Respondendo ao usuário
Eventos são ações que acontecem na página: click, keydown, submit, mouseover... Use addEventListener para "escutar" essas ações e executar código em resposta.
// HTML
<button id="meuBotao">Clique</button>

// JavaScript
let botao = document.querySelector('#meuBotao');

botao.addEventListener('click', function() {
  console.log('Clicou!');
});
Eventos comuns:
🖱️ click — clique do mouse
⌨️ keydown — tecla pressionada
📝 submit — envio de formulário
🖱️ mouseover — mouse passa por cima
📄 DOMContentLoaded — página carregada
2classList e criar elementos dinâmicos
Manipule classes CSS sem sobrescrever todas de uma vez. E crie elementos do zero com createElement e appendChild.
// Trabalhando com classes
elemento.classList.add('ativo');
elemento.classList.remove('inativo');
elemento.classList.toggle('dark');
elemento.classList.contains('ativo');
// Criando um elemento
let novoItem = document.createElement('li');
novoItem.textContent = 'Item novo';
lista.appendChild(novoItem);
4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Crie um arquivo HTML com <script> para cada exercício
Exercício 1 — Alterando texto
Crie um HTML com um <h1> e um <button>. Use querySelector e addEventListener para que, ao clicar no botão, o texto do h1 mude para "Título alterado!".
Exercício 2 — Mudando cores com classList
Crie um parágrafo e um botão. No CSS, crie uma classe .destaque com cor e fundo diferentes. Use classList.toggle para adicionar/remover essa classe ao clicar no botão.
Exercício 3 — Criando elementos
Crie uma <ul> vazia e um botão "Adicionar item". Use createElement e appendChild para adicionar um novo <li> com texto a cada clique.
Exercício 4 — Contador com eventos
Crie um contador com um <span> (começando em 0) e dois botões: "+" e "-". Use eventos de clique para aumentar ou diminuir o número exibido.
💡 Desafio extra: Crie uma lista de tarefas simples: um <input>, um botão "Adicionar" e uma <ul>. Ao clicar no botão, pegue o valor do input, crie um novo <li> e adicione à lista. Depois faça com que cada item da lista, ao ser clicado, seja removido (dica: use remove() no elemento clicado). Combine tudo que aprendeu: selecionar, eventos, criar elementos e remover.
