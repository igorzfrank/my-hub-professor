1 🔤 TIPOS PRIMITIVOS — A Matéria-Prima do Código
🧠 Todo valor tem um tipo
Em JavaScript, cada valor que você armazena em uma variável pertence a um tipo de dado. Os tipos primitivos são os mais fundamentais. Conhecer os tipos é essencial porque cada tipo se comporta de forma diferente com operadores e funções. Use o operador typeof para descobrir o tipo de qualquer valor.
🔤
String
Texto entre aspas simples, duplas ou crases. Ex: 'João', "olá", `Oi ${nome}`
🔢
Number
Números inteiros ou decimais. Ex: 42, 3.14, -10. Inclui NaN (não é número).
✅
Boolean
Apenas dois valores: true (verdadeiro) ou false (falso). Resultado de comparações.
// Três tipos primitivos principais
let nome = 'Maria';       // string
let idade = 30;         // number
let logado = true;      // boolean

// Descobrindo o tipo com typeof
console.log(typeof nome);   // "string"
console.log(typeof idade);  // "number"
console.log(typeof logado); // "boolean"
⚠️ Cuidado com o typeof
typeof null retorna "object" — isso é um bug histórico da linguagem. E typeof NaN retorna "number", mesmo significando "Not a Number". São peculiaridades que você precisa conhecer.
2 ➕ OPERADORES ARITMÉTICOS — Fazendo Contas
🧮 Operações matemáticas básicas
JavaScript funciona como uma calculadora. Você pode usar operadores aritméticos para realizar contas diretamente no código. O resultado pode ser armazenado em uma variável ou usado em outras operações.
+
Adição
-
Subtração
*
Multiplicação
/
Divisão
%
Módulo (resto)
**
Exponenciação
let a = 10;
let b = 3;

console.log(a + b);  // 13 (adição)
console.log(a - b);  // 7 (subtração)
console.log(a * b);  // 30 (multiplicação)
console.log(a / b);  // 3.333... (divisão)
console.log(a % b);  // 1 (resto da divisão)
console.log(a ** b); // 1000 (10 elevado a 3)
💡 O operador % (módulo): Muito útil para saber se um número é par (n % 2 === 0) ou ímpar (n % 2 === 1). Você usará isso em laços e condicionais nas próximas aulas.
3 🔗 CONCATENAÇÃO E OPERADORES DE COMPARAÇÃO
+Concatenando Strings
O operador + funciona de dois jeitos: com números ele soma, com strings ele concatena (junta). Template literals (crases) são a forma moderna de unir texto e variáveis.
let nome = 'Ana';
let idade = 25;

// Concatenação com +
console.log('Olá, ' + nome + '!');

// Template literal (recomendado)
console.log(`Olá, ${nome}! Você tem ${idade} anos.`);
?Operadores de Comparação
Comparam dois valores e retornam true ou false. Essenciais para tomadas de decisão no código.
==
Igual (valor)
===
Estritamente igual
!=
Diferente
!==
Estritamente diferente
>
Maior que
<
Menor que
>=
Maior ou igual
<=
Menor ou igual
console.log(10 > 5);   // true
console.log(10 === '10'); // false (number ≠ string)
console.log(10 == '10');  // true (conversão automática)
console.log(5 !== 3);   // true
⚠️ Regra de ouro: Use SEMPRE === e !== (com três sinais) em vez de == e !=. Os operadores triplos comparam valor E tipo, evitando bugs difíceis de encontrar.
4 ✏️ EXERCÍCIOS PRÁTICOS

Exercício 1 — Tipos e typeof
Crie uma variável de cada tipo primitivo (string, number, boolean). Use typeof para verificar e exibir o tipo de cada uma no console.
Exercício 2 — Calculadora simples
Crie duas variáveis numéricas. Calcule e exiba no console: soma, subtração, multiplicação, divisão e o resto da divisão entre elas.
Exercício 3 — Frase dinâmica
Crie variáveis para nome, idade e cidade. Use template literals (crases) para exibir: "Meu nome é [nome], tenho [idade] anos e moro em [cidade]."
Exercício 4 — Comparações
Teste comparações e exiba o resultado (true/false): 15 > 10, "10" === 10, 20 >= 20, 7 < 3, 5 !== 5. Explique cada resultado.
💡 Desafio extra: Crie uma variável preco = 150 e outra desconto = 20. Calcule o preço final com desconto (em reais) e exiba: "O preço com desconto é R$ [valor]." Depois verifique se o preço final é menor que 100 e exiba o resultado (true/false).
