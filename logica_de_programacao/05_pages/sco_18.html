Map, Filter e Find
15 exercícios · map · filter · find · Encadeamento · Imutabilidade
map filter find Prática
1-5 🗺️ MAP — TRANSFORMANDO ARRAYS
1
Dobrando números
Crie um array let numeros = [1, 2, 3, 4, 5, 6]. Use map para criar um novo array onde cada número é multiplicado por 2. Exiba o array original e o novo array.
2
Convertendo temperaturas
Crie um array let celsius = [0, 15, 30, 45]. Use map para converter para Fahrenheit (F = C * 9/5 + 32). Exiba o novo array com as temperaturas convertidas.
3
Transformando para maiúsculas
Crie um array let nomes = ['ana', 'bruno', 'carla', 'diego']. Use map para criar um novo array com todos os nomes em letras maiúsculas. Dica: use o método .toUpperCase().
4
Extraindo propriedades
Crie um array let produtos = [{nome: 'Mouse', preco: 50}, {nome: 'Teclado', preco: 120}, {nome: 'Monitor', preco: 800}]. Use map para criar um array apenas com os nomes dos produtos.
5
Adicionando prefixo
Crie um array let frutas = ['maçã', 'banana', 'uva']. Use map para criar um novo array onde cada fruta recebe o prefixo "Fruta: " (ex: "Fruta: maçã"). Exiba o resultado.
6-10 🔍 FILTER — FILTRANDO DADOS
6
Filtrando maiores que 10
Crie um array let numeros = [5, 12, 8, 21, 3, 15, 7]. Use filter para criar um novo array apenas com números maiores que 10. Exiba o resultado.
7
Filtrando números pares
Use o mesmo array do exercício 6. Crie um novo array usando filter que contenha apenas os números pares (dica: use o operador %). Exiba o resultado.
8
Filtrando por tamanho do texto
Crie um array let palavras = ['sol', 'montanha', 'lua', 'estrela', 'mar']. Use filter para retornar apenas palavras com mais de 3 letras. Dica: use .length.
9
Filtrando objetos por preço
Use o array produtos do exercício 4. Use filter para retornar apenas os produtos com preço maior que 60. Exiba o array resultante.
10
Filter que retorna array vazio
Usando o array numeros do exercício 6, use filter para encontrar números maiores que 100. O que é retornado? Exiba o resultado e verifique o .length do array resultante.
11-15 🎯 FIND E ENCADEAMENTO
11
Encontrando com find
Crie um array let idades = [12, 18, 25, 30, 15, 42]. Use find para encontrar o primeiro número maior que 20. Exiba o resultado. Depois teste com uma condição que não existe (ex: > 100).
12
Buscando por ID
Crie um array let usuarios = [{id: 1, nome: 'João'}, {id: 2, nome: 'Maria'}, {id: 3, nome: 'Pedro'}]. Use find para localizar o usuário com id igual a 2. Exiba o objeto encontrado.
13
Find em array de strings
Crie um array let frutas = ['maçã', 'banana', 'laranja', 'uva']. Use find para encontrar a primeira fruta que começa com a letra 'l'. Dica: use o método .startsWith().
14
Encadeando filter e map
Crie um array let valores = [10, 25, 8, 42, 15, 3, 30]. Encadeie filter e map em uma única linha: primeiro filtre os números maiores que 15, depois multiplique cada um por 3. Exiba o resultado.
15
Mini projeto: Catálogo inteligente
Crie um array let filmes = [{titulo: 'Matrix', ano: 1999, genero: 'Ficção'}, {titulo: 'Interestelar', ano: 2014, genero: 'Ficção'}, {titulo: 'Titanic', ano: 1997, genero: 'Romance'}, {titulo: 'Avatar', ano: 2009, genero: 'Ficção'}, {titulo: 'Coringa', ano: 2019, genero: 'Drama'}]. Depois: (1) use map para criar um array apenas com os títulos, (2) use filter para selecionar filmes do gênero 'Ficção', (3) use find para localizar o filme do ano de 1997, (4) encadeie filter e map para exibir apenas os títulos dos filmes lançados após o ano 2000.
💡 DICAS PARA RESOLUÇÃO
✅ O que você deve praticar
🗺️ Usar map para transformar cada item
🔍 Usar filter para selecionar por condição
🎯 Usar find para localizar um item único
🔗 Encadear filter().map() em uma linha
📦 Entender que os métodos não modificam o array original
🏹 Usar arrow functions como callback
⚠️ Erros comuns para evitar
❌ Achar que map modifica o array original
❌ Usar filter esperando modificar os itens (use map depois)
❌ Esquecer que find retorna undefined se não encontrar
❌ Não armazenar o retorno em uma variável
❌ Usar find esperando retornar todos os itens (use filter)
❌ Achar que forEach retorna algo (ele retorna undefined)
💡 Dica de ouro: map, filter e find são os métodos mais usados no desenvolvimento moderno com JavaScript. Dominar esses três vai facilitar muito sua vida ao trabalhar com APIs, listas de produtos, catálogos e qualquer tipo de dados estruturados. Pratique bastante! 🚀
