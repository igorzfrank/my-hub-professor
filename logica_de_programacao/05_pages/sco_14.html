Objeto Propriedades for...in Prática
1-5 🗂️ CRIAÇÃO E ACESSO A OBJETOS
1
Criando seu primeiro objeto
Crie um objeto chamado pessoa com as propriedades: nome (string), idade (number), altura (number) e estudante (boolean). Exiba o objeto completo no console.
2
Acessando com notação de ponto
Usando o objeto pessoa do exercício 1, acesse cada propriedade com a notação de ponto (pessoa.nome) e exiba frases como: "Nome: [valor]", "Idade: [valor]", etc.
3
Acessando com notação de colchetes
Acesse as mesmas propriedades do objeto pessoa usando a notação de colchetes (pessoa['nome']). Compare as duas formas. Crie uma variável let chave = 'idade' e use pessoa[chave] para acessar dinamicamente.
4
Objeto com diferentes tipos
Crie um objeto produto com: nome (string), preco (number), disponivel (boolean) e tags (array de strings). Use typeof para verificar o tipo de cada propriedade do objeto.
5
Template literals com objeto
Usando o objeto pessoa, crie uma frase completa usando template literals: "Meu nome é [nome], tenho [idade] anos, [altura]m de altura e sou estudante: [estudante]."
6-10 🔧 MANIPULANDO PROPRIEDADES
6
Adicionando propriedades
Crie um objeto carro com marca e modelo. Depois, adicione dinamicamente as propriedades: ano, cor e placa. Exiba o objeto antes e depois das adições.
7
Alterando valores
Usando o objeto carro, altere o ano para um valor mais recente e a cor para outra de sua preferência. Exiba o objeto resultante com as alterações.
8
Removendo com delete
Usando o objeto carro, remova a propriedade placa usando o operador delete. Exiba o objeto antes e depois. Verifique se a placa ainda existe com o operador in.
9
Verificando propriedades com in
Crie um objeto usuario com nome e email. Depois use o operador in para verificar se as propriedades 'nome', 'senha' e 'email' existem no objeto. Exiba true ou false para cada verificação.
10
Objeto dentro de objeto
Crie um objeto aluno com nome, idade e uma propriedade endereco que é outro objeto (rua, numero, cidade). Acesse e exiba: "O aluno mora na [rua], [numero] em [cidade]."
11-15 🔄 PERCORRENDO E ARRAYS DE OBJETOS
11
Primeiro for...in
Crie um objeto filme com titulo, diretor, ano e genero. Use for...in para percorrer todas as propriedades e exibir: "[chave]: [valor]".
12
Object.keys() na prática
Usando o objeto filme, use Object.keys() para obter um array com todas as chaves. Depois use Object.values() para obter um array com todos os valores. Exiba ambos.
13
Array de objetos simples
Crie um array amigos com 4 objetos, cada um com nome e idade. Use forEach para exibir: "Nome: [nome], Idade: [idade]".
14
Filtrando array de objetos
Usando o array amigos, crie um novo array apenas com as pessoas que têm idade maior ou igual a 18 anos. Percorra o array original com forEach e use if para decidir quem entra no novo array.
15
Mini projeto: Catálogo de livros
Crie um array biblioteca com 5 objetos representando livros (titulo, autor, paginas, lido). Depois: (1) use forEach para listar todos os títulos com autor, (2) use Object.keys no primeiro livro para ver suas propriedades, (3) use for...in no primeiro livro para exibir chave e valor, (4) verifique com in se a propriedade 'editora' existe em algum livro, (5) adicione a propriedade editora aos livros que não têm e exiba o array atualizado.
💡 DICAS PARA RESOLUÇÃO
✅ O que você deve praticar
🗂️ Criar objetos com chaves { }
🔑 Acessar com ponto e colchetes
➕ Adicionar novas propriedades dinamicamente
✏️ Alterar valores de propriedades existentes
🗑️ Remover com delete
✅ Verificar com operador in
🔄 Percorrer com for...in
📚 Trabalhar com arrays de objetos
🔧 Usar Object.keys() e Object.values()
⚠️ Erros comuns para evitar
❌ Esquecer as aspas nas chaves com colchetes: pessoa[idade] ❌ → pessoa['idade'] ✅
❌ Usar = em vez de : na criação do objeto
❌ Achar que objeto tem índice numérico como array
❌ Tentar usar forEach diretamente em objeto (só funciona em arrays)
❌ Esquecer que delete retorna true mesmo se a propriedade não existir
❌ Confundir for...in (objetos) com for...of (arrays)
💡 Dica de ouro: Arrays de objetos são o formato de dados mais comum no desenvolvimento web. Quando você consome uma API (Instagram, Spotify, etc.), os dados chegam exatamente assim: uma lista de objetos. Dominar arrays de objetos é essencial para qualquer projeto real! 🚀
