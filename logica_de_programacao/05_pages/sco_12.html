String Number Boolean Operadores
1-5 🔤 TIPOS DE DADOS E typeof
1
Identificando strings
Crie 3 variáveis com let armazenando textos diferentes: uma com aspas simples, outra com aspas duplas e outra com template literals (crases). Use typeof para verificar e exibir o tipo de cada uma no console.
2
Trabalhando com numbers
Crie variáveis para armazenar: um número inteiro, um número decimal, um número negativo e o valor NaN (use uma operação inválida como "abc" * 2). Use typeof em cada uma. O que o typeof retorna para NaN?
3
Booleanos na prática
Crie duas variáveis booleanas: let maiorDeIdade = true e let temFilhos = false. Use typeof para verificar o tipo. Depois crie uma terceira variável booleana que seja o resultado de uma comparação (ex: 10 > 5).
4
Misturando tipos
Crie uma variável para cada tipo (string, number, boolean). Depois, use console.log com typeof para cada uma, exibindo uma frase como: "O tipo da variável nome é string".
5
Descobrindo tipos especiais
Teste typeof com: null, undefined, [] (array vazio) e {} (objeto vazio). Anote os resultados. Qual deles retorna um tipo "errado" segundo a história do JavaScript?
6-10 ➕ OPERADORES ARITMÉTICOS E CONCATENAÇÃO
6
Mini calculadora
Crie duas variáveis let x = 50 e let y = 7. Calcule e exiba no console: soma, subtração, multiplicação, divisão, resto da divisão (%) e x elevado a y (**).
7
Par ou ímpar com módulo
Crie uma variável let numero = 17. Use o operador % para verificar se o número é par ou ímpar (exiba o resto da divisão por 2). Depois teste com outros valores: 24, 31, 100.
8
Concatenação com +
Crie variáveis let primeiroNome = "Ana" e let sobrenome = "Silva". Use o operador + para juntar os nomes com um espaço entre eles. Exiba o nome completo no console.
9
Template literals
Crie variáveis para: produto (string), quantidade (number) e preco (number). Use template literals (crases e ${}) para exibir: "Você comprou [quantidade] [produto](s) por R$ [preco * quantidade]."
10
O perigo do + com tipos mistos
Teste e explique o que acontece em cada caso: console.log(10 + 5), console.log("10" + 5), console.log(10 + "5"), console.log("10" + "5"). Por que os resultados são diferentes?
11-15 🔗 OPERADORES DE COMPARAÇÃO
11
Comparações básicas
Teste as seguintes comparações e anote se retornam true ou false: 10 > 5, 3 < 1, 7 >= 7, 9 <= 10, 4 == 4, 8 != 5. Exiba cada resultado com console.log e uma descrição.
12
== vs ===
Compare os resultados de: 5 == "5" vs 5 === "5", 0 == false vs 0 === false, null == undefined vs null === undefined. Qual é a diferença prática entre == e ===?
13
Comparando variáveis
Crie let idade = 18 e let idadeMinima = 21. Use operadores de comparação para verificar: se idade é maior que idadeMinima, se é igual, se é menor ou igual. Exiba os resultados booleanos.
14
Validação de formulário
Crie variáveis simulando um cadastro: let nome = "", let email = "joao@email.com", let senha = "123456". Use comparações para verificar: nome está vazio? (nome === ""), email contém "@"? (use .includes), senha tem mais de 5 caracteres? (senha.length > 5).
15
Mini projeto: Perfil de usuário
Crie um script completo com: (1) variáveis para nome, idade, altura, peso, (2) calcule o IMC (peso / (altura * altura)) e armazene em uma variável, (3) use comparações para verificar se a pessoa é maior de idade (>= 18), se o IMC está abaixo de 25 e se o nome tem mais de 3 letras. Exiba todos os resultados booleanos com frases descritivas usando template literals.
💡 DICAS PARA RESOLUÇÃO
✅ O que você deve praticar
🔤 Criar strings com aspas simples, duplas e crases
🔢 Trabalhar com números inteiros e decimais
✅ Criar e usar valores booleanos
🔍 Verificar tipos com typeof
➕ Usar operadores aritméticos (+, -, *, /, %, **)
🔗 Concatenar com + e template literals
⚖️ Comparar valores com ==, ===, !=, !==, >, <, >=, <=
⚠️ Erros comuns para evitar
❌ Usar + com string e number sem entender a conversão
❌ Confundir = (atribuição) com == ou === (comparação)
❌ Usar == em vez de ===
❌ Achar que typeof null retorna "null" (retorna "object")
❌ Esquecer que % retorna o resto, não a porcentagem
❌ Usar crase (\`) achando que é aspas simples (')
💡 Dica de ouro: No console do navegador, você pode testar comparações diretamente sem precisar criar variáveis. Digite 10 > 5 e veja o resultado na hora. Use isso para experimentar antes de escrever o código completo! 🚀
