Laços de Repetição — for, while e do...while
15 exercícios · for · while · do...while · break · continue · Percorrer arrays
for while break/continue Prática
1-5 🔄 FOR BÁSICO
1
Contagem crescente
Use um for para exibir no console os números de 1 a 10, um em cada linha. Depois faça o mesmo exibindo de 1 a 20.
2
Contagem decrescente
Use um for para exibir uma contagem regressiva de 10 até 1. Dica: comece com let i = 10, condição i >= 1 e decremente com i--.
3
Números pares com for
Use um for de 1 a 20. Dentro dele, use um if com o operador % para exibir apenas os números pares. Exiba também apenas os números ímpares em outro loop.
4
Soma de números
Use um for para somar todos os números de 1 a 100. Crie uma variável let soma = 0 antes do loop e vá somando dentro dele. Exiba o resultado final.
5
Tabuada com for
Crie uma variável let numero = 7. Use um for de 1 a 10 para exibir a tabuada completa: "7 x 1 = 7", "7 x 2 = 14"... até "7 x 10 = 70". Depois teste com outros números.
6-10 ⏳ while E do...while
6
Contagem regressiva com while
Use um while para fazer uma contagem regressiva de 10 até 1. Ao final do loop, exiba "Feliz Ano Novo!". Lembre-se de decrementar o contador dentro do loop.
7
Contagem crescente com while
Use um while para exibir os números de 1 a 15. Crie let i = 1 antes do loop e incremente com i++ dentro dele. A condição deve ser i <= 15.
8
Dobrando valores com while
Crie let valor = 2. Use um while para dobrar o valor (multiplicar por 2) enquanto ele for menor que 1000. Exiba cada valor no console. Quantas iterações foram necessárias?
9
do...while básico
Crie let x = 20. Use um do...while para exibir o valor de x e depois decrementar. A condição deve ser x > 10. Observe que o bloco executa pelo menos uma vez.
10
do...while com condição falsa
Crie let y = 10. Use um do...while com a condição y < 5. O que acontece? O bloco executa mesmo com a condição falsa? Explique a diferença para o while.
11-15 🛑 break, continue E ARRAYS
11
Break no for
Crie um for de 1 a 20. Use break para interromper o loop quando o número chegar a 13. Exiba todos os números exibidos antes da interrupção.
12
Continue no for
Crie um for de 1 a 15. Use continue para pular os números múltiplos de 5 (divisíveis por 5). Exiba apenas os números que não foram pulados.
13
Percorrendo array com for
Crie um array let frutas = ['maçã', 'banana', 'laranja', 'uva', 'morango']. Use um for clássico para percorrer e exibir cada fruta com seu índice: "Índice 0: maçã", "Índice 1: banana"...
14
Encontrando valores com break
Crie um array let numeros = [3, 8, 12, 5, 20, 7, 15]. Use um for para percorrer o array. Quando encontrar o número 20, use break para parar e exiba: "Número 20 encontrado na posição X".
15
Mini projeto: Análise de array
Crie um array let valores = [12, 5, 8, 21, 3, 14, 7]. Use um for para: (1) encontrar e exibir o maior número, (2) encontrar e exibir o menor número, (3) calcular e exibir a soma de todos os valores, (4) calcular e exibir a média (soma / quantidade). Dica: crie variáveis para maior, menor e soma antes do loop.
💡 DICAS PARA RESOLUÇÃO
✅ O que você deve praticar
🔄 Usar for com inicialização, condição e incremento
⏳ Usar while quando não sabe quantas repetições
🔁 Usar do...while para garantir ao menos uma execução
🛑 Usar break para interromper loops
⏭️ Usar continue para pular iterações
📚 Percorrer arrays com for usando .length
🧮 Acumular valores com variáveis antes do loop
⚠️ Erros comuns para evitar
❌ Criar loop infinito esquecendo de alterar a condição
❌ Usar while quando for seria mais adequado
❌ Esquecer de inicializar o contador antes do while
❌ Colocar break no lugar errado e sair antes da hora
❌ Confundir break (sai do loop) com continue (pula iteração)
❌ Usar índice errado ao percorrer array (esquecer que começa em 0)
❌ Não declarar variáveis de acúmulo (soma, maior) antes do loop
💡 Dica de ouro: Se você souber exatamente quantas vezes precisa repetir, use for. Se a repetição depende de uma condição que pode mudar, use while. Se precisar garantir pelo menos uma execução, use do...while. Com a prática, a escolha fica natural! 🚀
