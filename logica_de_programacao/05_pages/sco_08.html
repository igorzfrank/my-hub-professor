1 🧠 POR QUE USAR MAP, FILTER E FIND?
🧠 Além do forEach
O forEach executa uma ação para cada item, mas não retorna nada. Os métodos map, filter e find vão além: eles criam novos arrays ou retornam valores baseados no array original, sem modificá-lo. São métodos imutáveis — o array original permanece intacto. Isso é fundamental no desenvolvimento moderno.
📐Comparação visual dos três métodos
🗺️
map
Transforma cada item.
Retorna um novo array do mesmo tamanho.
[1, 2, 3] → [2, 4, 6]
🔍
filter
Filtra por condição.
Retorna um novo array menor ou igual.
[1, 2, 3, 4] → [2, 4]
🎯
find
Encontra por condição.
Retorna apenas um item (o primeiro).
[1, 2, 3, 4] → 3
🧠 Todos recebem uma função callback
Os três métodos funcionam de forma parecida: recebem uma função callback que é executada para cada item do array. Essa callback pode receber até 3 parâmetros: (item, índice, array) — mas o mais comum é usar apenas o primeiro.
2 🗺️🔍 map E filter EM AÇÃO
1map — Transformar cada item
Percorre o array, aplica uma função a cada item e retorna um novo array com os resultados. O array original não é modificado. Ideal para quando você precisa modificar todos os elementos.
let numeros = [1, 2, 3, 4, 5];

// Multiplicar todos por 2
let dobrados = numeros.map(num => num * 2);
console.log(dobrados); // [2, 4, 6, 8, 10]

// Extrair apenas nomes de objetos
let alunos = [
  { nome: 'Ana', nota: 8 },
  { nome: 'Bruno', nota: 6 }
];
let nomes = alunos.map(a => a.nome);
console.log(nomes); // ['Ana', 'Bruno']
2filter — Selecionar por condição
Percorre o array e retorna um novo array apenas com os itens que passaram no teste (condição). Se nenhum item passar, retorna um array vazio [].
let numeros = [10, 25, 8, 42, 15, 3];

// Filtrar apenas maiores que 20
let maiores = numeros.filter(n => n > 20);
console.log(maiores); // [25, 42]

// Filtrar alunos aprovados
let alunos = [
  { nome: 'Ana', nota: 8 },
  { nome: 'Bruno', nota: 5 }
];
let aprovados = alunos.filter(a => a.nota >= 7);
console.log(aprovados); // [{nome:'Ana', nota:8}]
💡 Encadeamento: Você pode encadear map e filter! Ex: array.filter(condicao).map(transformacao). Primeiro filtra, depois transforma o resultado — tudo em uma linha. Muito comum no dia a dia.
3 🎯 FIND — ENCONTRANDO UM ITEM ESPECÍFICO
1find — O primeiro que satisfaz a condição
Percorre o array e retorna o primeiro item que atende à condição. Se nenhum item for encontrado, retorna undefined. Diferente do filter (que retorna array), o find retorna o próprio item.
let numeros = [3, 7, 12, 9, 5];

let encontrado = numeros.find(n => n > 10);
console.log(encontrado); // 12

// Se não encontrar:
let inexistente = numeros.find(n => n > 100);
console.log(inexistente); // undefined
// Buscar por ID (muito comum)
let produtos = [
  { id: 1, nome: 'Mouse' },
  { id: 2, nome: 'Teclado' }
];
let produto = produtos.find(p => p.id === 2);
console.log(produto); // {id:2, nome:'Teclado'}

4 ✏️ EXERCÍCIOS PRÁTICOS
📝 Pratique no console ou em um arquivo .js
Exercício 1 — Dobrando valores com map
Crie um array precos = [10, 25, 40, 100]. Use map para criar um novo array com cada preço acrescido de 10% (multiplique por 1.1). Exiba o novo array.
Exercício 2 — Filtrando com filter
Usando o array do exercício 1, use filter para criar um novo array apenas com os preços maiores que 20. Exiba o resultado.
Exercício 3 — Buscando com find
Crie um array de objetos contatos com id e nome. Use find para buscar um contato pelo id. Exiba o objeto encontrado.
Exercício 4 — Encadeando map e filter
Dado o array numeros = [5, 12, 8, 21, 3, 15], primeiro filtre apenas os números maiores que 10 e depois multiplique cada um por 2. Faça tudo em uma linha encadeando filter e map.
💡 Desafio extra: Crie um array produtos com pelo menos 5 objetos (id, nome, preco, categoria). Depois: (1) use map para criar uma lista apenas com os nomes, (2) use filter para selecionar apenas produtos de uma categoria específica, (3) use find para localizar um produto pelo id e (4) encadeie filter e map para exibir apenas os nomes dos produtos acima de determinado preço.
