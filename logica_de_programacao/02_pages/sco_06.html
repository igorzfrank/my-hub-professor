🟢 Nível 1 — CASE + Variáveis e Cálculos (Desafios 1 a 3) escolha · caso · pare
Desafio 01Calculadora de Operações
Crie um programa que leia dois números reais e um código de operação (1 a 4). Use ESCOLHA...CASO para realizar a operação correspondente:
1 = Soma | 2 = Subtração | 3 = Multiplicação | 4 = Divisão
Exiba o resultado com duas casas decimais. Se o código for inválido, exiba "Operação inválida".
💡 Conceitos integrados: Declaração de variáveis (real, inteiro), leitura de dados, CASE para múltiplas operações, operações aritméticas.
Desafio 02Conversor de Unidades
Leia um valor real e um código de conversão (1 a 4). Use CASE para converter:
1 = Metros → Centímetros (× 100)
2 = Quilômetros → Metros (× 1000)
3 = Celsius → Fahrenheit (× 1.8 + 32)
4 = Reais → Dólares (÷ 5.50)
Exiba o valor original, a unidade de destino e o resultado.
💡 Conceitos integrados: Variáveis reais, CASE com cálculos diferentes em cada caso, exibição formatada de resultados.
Desafio 03Cardápio com Quantidade e Total
Exiba um menu com 5 produtos e preços. Leia o código do produto (1 a 5) e a quantidade desejada. Use CASE para identificar o preço unitário e calcule o valor total (preço × quantidade). Exiba o nome do produto, preço unitário, quantidade e total.
💡 Conceitos integrados: CASE para buscar preço, multiplicação para calcular total, exibição completa do resumo do pedido.
🟡 Nível 2 — CASE + SE + Operadores Lógicos (Desafios 4 a 5) se · senao · E · OU · NÃO
Desafio 04Sistema de Desconto por Categoria e Fidelidade
Leia o valor da compra, a categoria do cliente (1 a 4) e se é cliente fidelidade (S/N). Use CASE para definir o percentual de desconto:
1 = Comum (5%) | 2 = Bronze (10%) | 3 = Prata (15%) | 4 = Ouro (20%)

Regra adicional (use SE + E): Se for cliente fidelidade E a compra for maior que R$ 500, adicione mais 5% de desconto. Calcule e exiba o valor final com desconto.
💡 Conceitos integrados: CASE para percentual por categoria, SE com operador E para condição extra, cálculo de desconto acumulado.
Desafio 05Validador de Senha com CASE e Bloqueio
O sistema tem 3 senhas diferentes para 3 níveis de acesso. Leia o nível desejado (1 a 3) e a senha digitada. Use CASE para verificar a senha correta de cada nível:
1 = Usuário (senha "user123")
2 = Gerente (senha "admin456")
3 = Master (senha "master789")

Validação (use SE): O usuário tem apenas 3 tentativas no total (independente do nível). Se errar 3 vezes, exiba "Sistema bloqueado" e encerre.
💡 Conceitos integrados: CASE para múltiplas senhas, contador de tentativas, SE para validação, laço enquanto para repetir até acertar ou bloquear.
🔴 Nível 3 — CASE + Laços de Repetição (Desafios 6 a 7) para · enquanto · repita
Desafio 06Menu Interativo com Repetição
Crie um programa que exiba um menu com 5 opções e fique repetindo até o usuário digitar 0 (zero) para sair. Use CASE para cada opção:
1 = Exibir "Olá, Mundo!"
2 = Ler dois números e exibir a soma
3 = Ler um número e exibir se é par ou ímpar
4 = Ler um número e exibir a tabuada dele (use PARA)
5 = Exibir quantas vezes cada opção foi escolhida (use contadores)
0 = Sair do programa com mensagem de despedida
💡 Conceitos integrados: ENQUANTO para repetição do menu, CASE para executar cada opção, contadores acumulativos, PARA para tabuada.
Desafio 07Sistema de Votação Eletrônica
Simule uma urna eletrônica com 4 candidatos. Use um laço PARA repetir 10 vezes (10 eleitores). Em cada iteração, leia o voto (1 a 4). Use CASE para contabilizar:
1 = Candidato A | 2 = Candidato B | 3 = Candidato C | 4 = Candidato D
Qualquer outro número = Voto nulo (também contabilize)

Ao final, exiba o total de votos de cada candidato, os votos nulos e o nome do vencedor.
💡 Conceitos integrados: PARA com número fixo de repetições, CASE para classificar votos, acumuladores, SE para determinar o vencedor.
