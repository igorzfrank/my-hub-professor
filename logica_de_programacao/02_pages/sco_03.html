📌 Guia Rápido de Conversão — Pseudocódigo → Portugol
INÍCIO / FIM → funcao inicio() { }
LER → leia() (declare a variável antes com tipo)
ESCREVER → escreva()
← (atribuição) → =
SE...ENTÃO...SENÃO...FIM_SE → se ( ) { } senao { }
PARA...FAÇA...FIM_PARA → para ( ; ; ) { }
MOD → %
= (comparação) → ==
⚠️ Lembrete: No Portugol, declare o tipo de cada variável antes de usar (inteiro, real, cadeia, logico). Não use ponto e vírgula no final dos comandos. Não esqueça as aspas duplas em textos e os parênteses nas condições do se.
