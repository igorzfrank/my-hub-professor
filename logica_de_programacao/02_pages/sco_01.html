1 🧠 O que é Portugol?
🎯 Definição
Portugol é uma linguagem de programação educacional baseada em português estruturado. Diferente do pseudocódigo (que é apenas um rascunho), o Portugol pode ser digitado, executado e testado no Portugol Studio, permitindo ver o algoritmo funcionando na prática.
💡 Analogia simples
Se o pseudocódigo é o rascunho no papel de uma casa (você desenha como quiser), o Portugol é a maquete 3D — você pode andar por ela, testar se as portas abrem e ver se tudo funciona antes de construir de verdade.

3 🏗️ Estrutura Básica e Primeiro Programa
📐Anatomia de um programa em Portugol
programa {
  funcao inicio() {
    escreva("Olá, Mundo!")
  }
}
🧠 Explicação linha a linha
programa { } → contêiner obrigatório que envolve todo o código.
funcao inicio() { } → ponto de entrada do programa. Toda execução começa aqui.
escreva("texto") → imprime uma mensagem na tela do usuário.
Aspas duplas → obrigatórias para textos (strings).
Minúsculas → comandos em minúsculo: escreva, funcao, inicio.
💡 Observação: Opcionalmente, pode-se incluir a biblioteca padrão na primeira linha: inclua biblioteca Biblioteca — necessário quando se usa recursos como limpa(), sorteia() ou timer().
⚠️ Atenção: Se esquecer de fechar as aspas " ou as chaves { }, o Portugol Studio acusará erro de sintaxe e o programa não executará. Diferente de linguagens como Java ou C#, o Portugol não exige ponto e vírgula ao final dos comandos.
4 ⭐ Regras de Ouro do Portugol
1Estrutura obrigatória: funcao inicio() { }
Todo programa Portugol precisa de uma função inicio() dentro de programa { }. É a partir dela que a execução começa. Sem essa função, o Portugol Studio não saberá por onde começar.
2Comandos em minúsculo e sem ponto e vírgula
Todas as palavras reservadas são escritas em minúsculas: se, senao, para, enquanto, escreva, leia, funcao. Diferente de Java ou C#, não se usa ponto e vírgula no final dos comandos — cada linha já é um comando.
3Blocos delimitados por chaves { }
No Portugol, blocos de código são delimitados por chaves { }. Diferente do pseudocódigo que usa palavras como FIM_SE e FIM_PARA, no Portugol o fechamento é sempre com }.
4Variáveis precisam ser declaradas com tipo
Antes de usar uma variável, declare seu tipo: inteiro idade, real preco, cadeia nome, logico ativo. No pseudocódigo, isso não é obrigatório.
5Indentação (mesmo princípio do pseudocódigo)
Recue o código para a direita quando estiver dentro de um bloco (se, para, enquanto). Isso não é obrigatório para execução, mas é essencial para legibilidade.
5 🔄 Tradução Lado a Lado — Mesmo Algoritmo em Duas Linguagens
👋 Exemplo 1 — Programa mais simples
📝 Pseudocódigo

INÍCIO
  ESCREVER "Olá, Mundo!"
FIM
💻 Portugol

programa {
  funcao inicio() {
    escreva("Olá, Mundo!")
  }
}
🔍 Comparação: No pseudocódigo, INÍCIO/FIM delimitam o bloco principal. No Portugol, funcao inicio() { } cumpre esse papel — é o ponto de partida da execução. Note que não há ponto e vírgula após o comando escreva().
🔢 Exemplo 2 — Verificar se um número é par ou ímpar
📝 Pseudocódigo

INÍCIO
  LER numero
  SE numero MOD 2 = 0 ENTÃO
    ESCREVER "PAR"
  SENÃO
    ESCREVER "ÍMPAR"
  FIM_SE
FIM
💻 Portugol

programa {
  funcao inicio() {
    inteiro numero
    leia(numero)
    se (numero % 2 == 0) {
      escreva("PAR")
    } senao {
      escreva("ÍMPAR")
    }
  }
}
🔍 Principais diferenças: Variável declarada com tipo (inteiro), % em vez de MOD, parênteses na condição, chaves { } no lugar de ENTÃO/FIM_SE. E sem ponto e vírgula no final dos comandos.
🔁 Exemplo 3 — Contar de 1 até 5
📝 Pseudocódigo

INÍCIO
  PARA i DE 1 ATÉ 5 FAÇA:
    ESCREVER i
  FIM_PARA
FIM
💻 Portugol

programa {
  funcao inicio() {
    inteiro i
    para (i = 1; i <= 5; i++) {
      escreva(i)
    }
  }
}
🔍 Principais diferenças: O laço PARA no Portugol usa parênteses com três partes: inicialização, condição e incremento. i++ significa i = i + 1. Comandos sem ponto e vírgula.
🎓 Exemplo 4 — Calcular média e verificar aprovação
📝 Pseudocódigo

INÍCIO
  LER nota1, nota2, nota3
  media ← (nota1 + nota2 + nota3) / 3
  SE media >= 7 ENTÃO
    ESCREVER "Aprovado: ", media
  SENÃO
    ESCREVER "Reprovado: ", media
  FIM_SE
FIM
💻 Portugol

programa {
  funcao inicio() {
    real nota1, nota2, nota3, media
    leia(nota1, nota2, nota3)
    media = (nota1 + nota2 + nota3) / 3
    se (media >= 7.0) {
      escreva("Aprovado: ", media)
    } senao {
      escreva("Reprovado: ", media)
    }
  }
}
🔍 Principais diferenças: Variáveis declaradas com tipo real. Operador de atribuição é = (não ←). Múltiplas variáveis podem ser declaradas e lidas na mesma linha separadas por vírgula. Sem ponto e vírgula.
6 📦 Variáveis e Tipos de Dados no Portugol
🎯 Diferença crucial
No pseudocódigo, você pode simplesmente escrever LER nome sem declarar a variável antes. No Portugol, toda variável precisa ser declarada com seu tipo antes de ser usada — isso é uma exigência da linguagem.
📂Tipos de dados no Portugol
inteiro
Números sem vírgula
Ex: 42, -7, 0
real
Números com vírgula
Ex: 3.14, -0.5, 7.0
cadeia
Texto (string)
Ex: "Maria", "ABC"
logico
Verdadeiro ou falso
Ex: verdadeiro, falso
// Declarando variáveis antes de usar
inteiro idade
real altura
cadeia nome
logico ativo

// Atribuindo valores (sem ponto e vírgula)
idade = 25
altura = 1.75
nome = "Carlos"
ativo = verdadeiro
✅ Observação importante: Note que nas atribuições acima não há ponto e vírgula. No Portugol Studio, cada linha de comando já é naturalmente finalizada pela quebra de linha — o ponto e vírgula não é necessário e pode até causar erro de sintaxe em algumas versões.
7 🎯 Quando Usar Cada Um?
📝 Use Pseudocódigo quando...
✅ Estiver aprendendo lógica pela primeira vez
✅ Estiver fazendo prova ou exercício no papel
✅ Precisar rascunhar um algoritmo rapidamente
✅ Quiser discutir a lógica com colegas
✅ Não tiver um computador por perto
✅ O foco for o raciocínio, não a sintaxe
💻 Use Portugol quando...
✅ Quiser ver o algoritmo funcionando de verdade
✅ Precisar testar se a lógica está correta
✅ Estiver aprendendo sintaxe de programação
✅ Quiser depurar (encontrar erros) no código
✅ Precisar mostrar resultado para alguém
✅ Estiver migrando para linguagens reais (Java, C, Python)

💡 Dica importante: Ao migrar do Portugol para linguagens como Java ou C#, você precisará se adaptar ao uso de ponto e vírgula no final de cada comando — algo que o Portugol não exige. Essa é uma das principais diferenças sintáticas que você encontrará nessa transição.
