1 📘 O que é Portugol?
Pseudocódigo vs. Portugol — Qual a diferença?
Pseudocódigo é um rascunho livre — você pode escrever de qualquer jeito, mas ele não executa.
Portugol (com Portugol Studio) é uma linguagem real, com sintaxe rigorosa, que você pode digitar, executar e testar.
⚙️Regras de Ouro do Portugol
1
📍 Estrutura obrigatória
Todo programa começa com programa { } e termina com um ponto final. O código executável vai dentro das chaves.
2
💬 Comandos em minúsculo
Use escreva, leia, se, senao, para, faca. Todo comando termina com ponto e vírgula.
3
↩️ Identação (mesmo princípio)
Assim como no pseudocódigo, recue o código para a direita sempre que ele estiver dentro de um bloco (se, para, enquanto).
2 🔨 Sua Primeira Linha de Código
Exemplo 1 — "Olá, Mundo!"
programa
{
  escreva("Olá, Mundo!");
}
🧠 O que está acontecendo?
programa { } — contêiner obrigatório do código.

escreva("texto"); — imprime uma mensagem na tela.

; — ponto e vírgula encerra cada comando.

Dica: Sempre feche as aspas duplas e não esqueça o ponto e vírgula!
3 ⚡ Condição e Repetição no Portugol
🔀Condicional — se ... senao
programa
{
  real nota
  leia(nota);
  se (nota >= 6.0) {
    escreva("Aprovado");
  } senao {
    escreva("Reprovado");
  }
}
🧠 Diferenças para o pseudocódigo
• Em vez de ENTÃO, usamos { } (chaves).
• A condição fica entre parênteses.
• Em vez de SENÃO, escreve-se senao (junto, minúsculo).
• Não existe FIM_SE — o bloco termina com }.
🔁Repetição — para ... faca
programa
{
  inteiro i
  para (i = 1; i <= 5; i++) {
    escreva("Olá");
  }
}
🧠 O que mudou?
• O laço usa parênteses com três partes:
  ① inicial (i = 1)
  ② condição (i <= 5)
  ③ incremento (i++)
• i++ significa i = i + 1.
• Bloco delimitado por chaves { }.
4 🧪 Prática Guiada — Exemplos Completos
A 🔢 Par ou Ímpar (com módulo %)
programa
{
  inteiro num
  leia(num);
  se (num % 2 == 0) {
    escreva("É PAR");
  } senao {
    escreva("É ÍMPAR");
  }
}
🧠 Observações
• inteiro declara a variável.
• % é o operador de resto (módulo).
• == significa "igual a". Um = só é usado para atribuir valor.
B ✖️ Tabuada de um número
programa
{
  inteiro num, i
  leia(num);
  para (i = 1; i <= 10; i++) {
    escreva(num, " x ", i, " = ", num * i);
  }
}
🧠 Detalhes importantes
• Duas variáveis declaradas na mesma linha: inteiro num, i.
• escreva aceita vários argumentos separados por vírgula.
• Expressão matemática diretamente dentro do escreva.
C ➕ Soma dos números de 1 a N
programa
{
  inteiro n, i, soma
  soma = 0;
  leia(n);
  para (i = 1; i <= n; i++) {
    soma = soma + i;
  }
  escreva("Soma: ", soma);
}
🧠 Padrão acumulador
• soma = 0 inicializa o acumulador.
• Dentro do laço: soma = soma + i acumula.
• Ao final, exibe o resultado fora do laço.
• Esse padrão é usado em médias, totais e contagens.
🏆 Desafios de Programação
1 🔐 Maioridade
Desafio: Crie um programa em Portugol que leia a idade de uma pessoa e escreva "Maior de idade" se tiver 18 anos ou mais, e "Menor de idade" caso contrário.

Dica: Use inteiro idade, leia(idade) e se (idade >= 18).
2 🔁 Contagem Regressiva
Desafio: Escreva um programa que leia um número inteiro N e exiba uma contagem regressiva de N até 0 (inclusive o zero). Use para ... faca.

Exemplo: Se N = 5 → saída: 5 4 3 2 1 0.
3 📊 Média de Notas
Desafio: Leia 3 notas de um aluno (tipo real), calcule a média aritmética e mostre se o aluno foi Aprovado (média ≥ 7.0) ou Reprovado.
4 ⭐ BÔNUS — Números Pares em um Intervalo
Desafio: Leia dois números inteiros A e B (com A menor que B) e exiba apenas os números pares entre A e B (incluindo os extremos, se forem pares).

Dica: Combine para com se (numero % 2 == 0) dentro do laço.
