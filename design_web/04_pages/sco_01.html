1 🎯 O que são Seletores CSS?
🎯 Definição
Seletores são padrões usados para selecionar e direcionar elementos HTML específicos para aplicar estilos. Eles definem "quem" vai receber as regras de estilo que você escreve.
💡 Analogia simples
Se o HTML é um fichário com muitas fichas (elementos), o seletor é como um filtro de busca – você pode buscar por nome (tag), por etiqueta colorida (classe), por identificador único (ID), ou pela posição na gaveta (relacionamentos).
📂Tipos de Seletores Básicos
🏷️ Seletor de Tipo (Tag)
Seleciona todos os elementos de um tipo. Ex: p { } afeta TODOS os parágrafos.
🎨 Seletor de Classe ( . )
O mais versátil. Ex: .destaque { } afeta todos com class="destaque".
🔢 Seletor de ID ( # )
Para elemento único. Ex: #cabecalho { }. Não pode repetir na página.
⭐ Seletor Universal ( * )
Seleciona TODOS os elementos. Ex: * { margin: 0; } é comum para reset.
2 🧪 Seletores na Prática — Exemplos Lado a Lado
🏷️ Seletor de Tipo (Tag)
/* CSS */
p { color: blue; }
h1 { font-size: 32px; }
button { background: green; }
<!-- HTML -->
<p>Fica azul</p>
<h1>32px</h1>
<h2>Não afetado</h2>
🎨 Seletor de Classe ( . ) — O mais usado
/* CSS */
.destaque { background: yellow; }
.botao-primario { background: blue; }
<!-- HTML -->
<p class="destaque">Amarelo</p>
<p>Não afetado</p>
🔢 Seletor de ID ( # ) — Elemento único
/* CSS */
#cabecalho { background: #333; }
<!-- HTML -->
<div id="cabecalho">Único</div>
⭐ Seletor Universal ( * ) — Reset global
/* CSS */
* {
  margin: 0;
  padding: 0;
}
<!-- Afeta TODOS os elementos -->
Remove margens e paddings padrão de tudo.
3 🎭 Pseudo-classes e Pseudo-elementos
🎭 Pseudo-classes — Estados e Posições
Selecionam elementos em estados específicos (mouse sobre, clicado, focado) ou com posições especiais (primeiro, último, ímpares).
a:hover { color: red; }
Mouse sobre o link
button:active { scale: 0.95; }
Botão sendo pressionado
input:focus { outline: blue; }
Campo selecionado
li:first-child { bold; }
Primeiro item da lista
li:nth-child(odd) { bg: gray; }
Itens ímpares (1, 3, 5...)
input:required { border: orange; }
Campo obrigatório
input:checked { bg: green; }
Checkbox/radio marcado
button:disabled { bg: gray; }
Botão desabilitado
✨ Pseudo-elementos — Partes específicas de um elemento
Selecionam partes específicas de um elemento que não existem no HTML, como a primeira letra ou um conteúdo decorativo.
p::first-letter { font-size: 2em; }
Primeira letra do parágrafo
p::first-line { bold; }
Primeira linha do parágrafo
::selection { bg: yellow; }
Texto selecionado pelo mouse
.icone::before { content: "⭐"; }
Insere conteúdo antes do elemento
.icone::after { content: "→"; }
Insere conteúdo depois do elemento
4 📐 Propriedades CSS — O que você pode estilizar
🎯 O que são Propriedades CSS?
Propriedades CSS são características ou estilos que você pode aplicar aos elementos selecionados. Cada propriedade define um aspecto visual específico (cor, tamanho, posição, comportamento). Se os seletores são "quem", as propriedades são "o quê" você vai mudar.

/* Exemplo completo: várias propriedades em um elemento */
.card {
  width: 300px;
  padding: 20px;
  margin: 10px;
  background-color: #f5f5f5;
  border: 1px solid #ccc;
  border-radius: 8px;
  font-family: 'Arial', sans-serif;
  text-align: center;
}
5 📦 BOX MODEL — O Conceito Fundamental do CSS
🎯 O que é o Box Model?
O Box Model é o conceito fundamental do CSS que define que todo elemento HTML é uma caixa retangular composta por quatro camadas concêntricas: conteúdo (content), preenchimento (padding), borda (border) e margem (margin).
💡 Analogia simples — Uma foto emoldurada
Conteúdo = a foto em si
Padding = espaço entre a foto e a moldura (espuma protetora)
Borda = a moldura da foto
Margin = espaço entre a moldura e a parede da sala
📐As 4 camadas do Box Model
📄
1. Conteúdo
Texto, imagem ou elemento interno
width / height
🧽
2. Padding
Espaço interno entre conteúdo e borda
padding
🖼️
3. Borda
Linha que envolve o padding
border
📏
4. Margin
Espaço externo entre elementos
margin
🧮 Como calcular o tamanho total de um elemento
Largura Total = width + padding-left + padding-right + border-left + border-right
Altura Total = height + padding-top + padding-bottom + border-top + border-bottom

/* A margem NÃO entra no tamanho do elemento — é espaço externo */
💡 Prática recomendada: box-sizing: border-box
Por padrão, padding e border aumentam o tamanho do elemento. Com box-sizing: border-box, o tamanho definido inclui padding e border, facilitando muito o layout.
/* Aplique globalmente no início do seu CSS */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
