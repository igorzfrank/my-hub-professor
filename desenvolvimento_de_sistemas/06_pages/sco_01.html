Comunicação entre React e Mundo Exterior

Entenda como o React gerencia o envio e recebimento de dados sem interromper a experiência do usuário.

🍽️1. Fundamentos: API REST e HTTP
Uma API REST funciona como um garçom: você (Cliente) faz um pedido, ele leva à cozinha (Servidor/Banco) e traz a resposta. Toda essa conversa usa o protocolo HTTP.
Verbo Ação Exemplo
GET Leitura Listar produtos.
POST Criação Cadastrar usuário.
PUT Atualização Alterar senha.
DELETE Exclusão Remover do carrinho.
2 🚀 Fetch API vs. Axios

Embora o fetch() seja nativo, o Axios é o padrão de mercado pela sua robustez e facilidade.

// Instalação
npm install axios

// services/api.js
import axios from 'axios';
const api = axios.create({ baseURL: 'https://api.exemplo.com' });
⚡3. Hooks de Efeito: useEffect

O useEffect gerencia efeitos colaterais. Buscar dados é um efeito colateral, pois muda o estado após o componente nascer.

useEffect(() => { loadData(); }, []); // O [] garante que rode só uma vez.
4 💻 Prática: Listando Usuários

import React, { useState, useEffect } from 'react';
import api from './services/api';

function UserList() {
  const [users, setUsers] = useState([]);

  useEffect(() => {
    // Exemplo de boa prática: função assíncrona dentro do useEffect
    async function fetchUsers() {
      try {
        const response = await api.get('/users');
        setUsers(response.data); // Armazena os dados no estado
      } catch (error) {
        console.error("Erro ao carregar usuários", error);
      }
    }

    fetchUsers();
  }, []);

  return (
    <div style={{ padding: '20px' }}>
      <h2>👥 Lista de Usuários (API)</h2>
      <ul>
        {users.map(user => (
          <li key={user.id}>
            <strong>{user.name}</strong> - {user.email}
          </li>
        ))}
      </ul>
    </div>
  );
}

export default UserList;
