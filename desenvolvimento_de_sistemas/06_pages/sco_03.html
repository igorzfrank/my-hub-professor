🍕 Sistema Pizzaolo
Cardápio Digital Dinâmico
Consumindo e Renderizando Dados Reais

Objetivo: Integrar o React com uma API própria para listar sabores, preços e categorias de pizzas em tempo real.

1 🔗 Configuração do Endpoint

No arquivo src/services/api.js, configure a URL base gerada pelo MockAPI:

para utilizar o Axios, no terminal você precisa instalar a dependência: npm install npm install axios.

import axios from 'axios';

const api = axios.create({
  baseURL: 'https://69ea53d715c7e2d51269b1cd.mockapi.io/pizzaolo/'
});

export default api;
2 💻 Componente Cardapio.jsx

Implemente o Hook useEffect para buscar os dados assim que o componente for montado.

useEffect(() => {
  async function obterDados() {
    try {
      const res = await api.get('/pizza'); // Rota do MockAPI
      setListaPizzas(res.data);
    } catch (erro) {
      console.error("Falha no cardápio", erro);
    } finally {
      setCarregando(false);
    }
  }
  obterDados();
}, []);
🧐Ponto de Atenção: Debugging
O nome do recurso deve ser idêntico ao criado no MockAPI.
✅ api.get('/pizza') | ❌ api.get('/pizzas') (Gera Erro 404)
Dica: Abra o F12 > Network, clique na requisição 'pizza' e veja o Preview. Se o JSON aparecer lá, seu front-end está pronto para mapeá-lo!
🚀 O Poder da API
Ao alterar um preço no MockAPI, a interface do React atualiza sozinha no próximo F5. Isso é desacoplamento!
