O que são Props no React?

📦 Comunicação entre componentes · 🔁 Fluxo Pai → Filho

 

📌 Definição

 

Props (properties) são dados que um componente recebe de outro componente.

 

👉 Permitem que componentes sejam dinâmicos e reutilizáveis.

🧠 Analogia Simples

 

f(x) = x + 1

 

function Saudacao(props) {
  return <h1>Olá, {props.nome}</h1>;
}

📦 Exemplo Básico

 

function Saudacao(props) {
  return <h2>Olá, {props.nome}!</h2>;
}

 

function App() {
  return <Saudacao nome="Keliven" />;
}

✨ Desestruturação (Forma Moderna)

 

function Saudacao({ nome }) {
  return <h2>{nome}</h2>;
}

🔁 Props com map()

 

const alunos = [
  { nome: "Ana", idade: 17 },
  { nome: "João", idade: 18 }
];

{alunos.map((aluno, index) => (
  <Card key={index} nome={aluno.nome} idade={aluno.idade} />
))}

⚠️ Regra Importantíssima

 

Props são somente leitura. Nunca altere props dentro do componente.

 

props.nome = "Outro nome"; // ❌ ERRADO

📊 Fluxo das Props: App.jsx → envia nome → Saudacao.jsx → usa {nome} → Tela
