Iniciando um Projeto com Vite + React

⚛️ Setup inicial · 🧩 Componentes · 🧠 Conceitos fundamentais

 

📂 1 — Criando o Projeto

 

1️⃣ Criar uma pasta para o projeto

2️⃣ Abrir a pasta no Visual Studio Code

3️⃣ Abrir o terminal (CTRL + J ou CTRL + ')

 

💻 Comando inicial:

 

npm create vite@latest

 

➡ Digite o nome do projeto e pressione Enter

➡ Escolha React com as setas e pressione Enter

➡ Escolha JavaScript e pressione Enter

➡ Confirme (y) pressionando Enter

 

📦 Rodando o projeto

 

cd nome-do-projeto
npm install
npm run dev

 

🔗 Após rodar, segure CTRL e clique no link exibido no terminal para abrir no navegador.

🧠 Entendendo o vite@latest

 

vite → nome do pacote

@latest → versão mais recente disponível

 

Ou seja: você está criando o projeto usando a versão mais atual do Vite 🚀

🧩 Componentes React

 

Componentes são funções ou classes que retornam elementos React.

 

Permitem combinar HTML + CSS + JavaScript em blocos reutilizáveis 🧱

 

Um botão, formulário, tela ou navbar: tudo pode ser um componente.

🏗️ Componentes de Classe

 

São classes que estendem React.Component e possuem método render().

 

✔ Possuem estado interno (state)

✔ Usam ciclo de vida (componentDidMount etc.)

✔ Indicados para lógica mais complexa

⚡ Componentes Funcionais

 

São funções JavaScript que retornam JSX.

 

✔ Sintaxe mais simples

✔ Podem usar Hooks (useState, useEffect)

✔ Recomendados atualmente

 

💡 Hoje, a preferência é usar componentes funcionais sempre que possível, pois oferecem código mais limpo e melhor reutilização de lógica.
