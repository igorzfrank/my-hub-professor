🌐 1. Conceito de SPA (Single Page Application)

 

Uma SPA é uma aplicação web que carrega uma única página HTML e atualiza dinamicamente o conteúdo conforme o usuário interage 🔁.

 

Diferente da MPA, onde cada clique carrega uma nova página do servidor.

🔄 Fluxo de funcionamento de uma SPA

 

 
 

🧠 Interpretação do fluxo:

1) O navegador carrega um único HTML

2) O JavaScript assume o controle

3) As interações atualizam apenas partes da tela

4) Dados são buscados via API (sem recarregar página)

 

🧩 Vantagens da SPA

 

✔️ Experiência semelhante a apps mobile 📱

✔️ Melhor performance após carregamento inicial ⚡

✔️ Separação clara entre Front-end e Back-end 🔌

✔️ Reutilização de componentes 🧱

⚛️ 2. O que é React?

 

O React é uma biblioteca JavaScript para construção de interfaces de usuário. Foi criada pela Meta (antigo Facebook) 🏢.

 

📌 Definição

 

React JS é uma biblioteca JavaScript usada para criar interfaces de usuário (UI), principalmente para aplicações web modernas e dinâmicas.

 

Ele é muito utilizado para construir SPAs (aplicações que não recarregam a página inteira a cada clique) 🚀

 

Foi criado pela Meta (antigo Facebook) e é uma das tecnologias mais usadas no front-end.

 

 

 

🧠 Para que serve o React?

 

✔ Criar interfaces interativas

✔ Organizar o código em componentes reutilizáveis

✔ Atualizar a tela de forma rápida e eficiente

✔ Trabalhar com aplicações que consomem APIs 🔌

 

📌 Biblioteca ou framework?

 

➡️ Biblioteca (foco em UI)

➡️ Não impõe arquitetura completa

➡️ Pode ser combinado com outras ferramentas

🏗️ Princípios do React

 

1️⃣ Componentização (blocos reutilizáveis)

2️⃣ Virtual DOM (otimiza atualizações)

3️⃣ Dados unidirecionais (pai → filho)

 

📊 DOM vs Virtual DOM

 

 
 

🧠 O que acontece quando algo muda?

1) O estado muda

2) O React cria um novo Virtual DOM

3) Compara com o anterior (reconciliation)

4) Atualiza apenas o que mudou ✅

🧩 Como o React funciona?

 

1️⃣ Componentes

 

A interface é dividida em blocos reutilizáveis (cada parte da tela pode virar um componente).

 

function Botao() {
  return <button>Clique aqui</button>
}

 

Exemplos de componentes: Navbar, Card, Botão, Formulário, Footer.

2️⃣ JSX

 

JSX permite escrever algo parecido com HTML dentro do JavaScript.

 

const titulo = "Olá Mundo"

return <h1>{titulo}</h1>

🌐 React é framework?

 

❌ Não. ✔ O React é uma biblioteca (foco em UI). Você pode combinar com outras ferramentas para criar aplicações completas.

📊 React é usado onde?

 

📱 Instagram

👥 Facebook

🎬 Netflix

💬 WhatsApp Web

🎯 Resumindo: React JS é 🧱 uma biblioteca para criar interfaces, 🧩 baseada em componentes, ⚡ usa Virtual DOM para performance e 🌐 é muito usada em SPAs ✅

📂 4. Estrutura de Pastas

 

Estrutura padrão do projeto:

 

meu-projeto/
├── node_modules/
├── public/
├── src/
│  ├── assets/
│  ├── App.jsx
│  ├── main.jsx
│  └── index.css
├── package.json
└── vite.config.js

 

🔹 main.jsx

 

Ponto de entrada da aplicação. Responsável por renderizar o App.

🔹 App.jsx

 

Componente principal. A aplicação começa nele e vai chamando os componentes filhos.

🔄 5. Fluxo de Renderização

 

📍 main.jsx (exemplo)

 

import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App'

ReactDOM.createRoot(document.getElementById('root')).render(
  <App />
)

 

🧠 Interpretação:

1) Busca o elemento root no HTML

2) Renderiza o componente App

3) Tudo começa a partir dele ✅

🧬 6. Introdução ao JSX

 

JSX = JavaScript + XML. Permite escrever “HTML” dentro do JavaScript 🧩.

 

🧠 Exemplo

 

function App() {
  return (
    <div>
      <h1>Olá React!</h1>
      <p>Minha primeira aplicação.</p>
    </div>
  )
}

⚠️ Regras do JSX

 

✔ Deve ter um elemento pai

✔ class → className

✔ for → htmlFor

✔ Expressões JS entre { }

 

🧩 Exemplo com expressão

 

function App() {
  const nome = "Keliven"

  return <h1>Olá, {nome}!</h1>
}

🧱 7 — Primeiros Componentes Funcionais

 

Um componente é uma função que retorna JSX.

 

🔹 Criando componente

 

function Saudacao() {
  return <h2>Bem-vindo ao React!</h2>
}

🔹 Usando componente

 

function App() {
  return (
    <div>
      <Saudacao />
    </div>
  )
}
