1. Métodos para Listas (Arrays)
Os arrays possuem métodos embutidos poderosos que permitem adicionar ou remover elementos tanto no início quanto no final da lista. Conhecer esses métodos é fundamental para gerenciar filas e pilhas de dados.
 
 
 
let tarefas = ["Estudar HTML", "Estudar CSS"];
 
// PUSH: Adiciona um ou mais elementos ao FINAL da lista
tarefas.push("Estudar JS");
console.log(tarefas); // ["Estudar HTML", "Estudar CSS", "Estudar JS"]
 
// POP: Remove e retorna o ÚLTIMO elemento da lista
let ultimaTarefa = tarefas.pop();
console.log(ultimaTarefa); // "Estudar JS"
 
// UNSHIFT: Adiciona elementos no INÍCIO da lista
tarefas.unshift("Configurar Ambiente");
 
// SHIFT: Remove e retorna o PRIMEIRO elemento da lista
let primeiraTarefa = tarefas.shift();
2. Manipulação de Dicionários (Objetos)
Diferente das listas onde usamos métodos, a manipulação básica de dicionários (Objetos) ocorre diretamente nas propriedades. No entanto, o JavaScript fornece a classe Object com métodos estáticos utilitários para extrair chaves e valores.
 
 
 
const usuario = { nome: "Carlos", perfil: "Admin" };
 
// Adicionando ou atualizando uma nova propriedade
usuario.email = "carlos@email.com";
 
// DELETE: Removendo uma propriedade do objeto
delete usuario.perfil;
 
// OBJECT.KEYS: Retorna uma lista (array) apenas com as chaves
const chaves = Object.keys(usuario);
console.log(chaves); // ["nome", "email"]
 
// OBJECT.VALUES: Retorna uma lista (array) apenas com os valores
const valores = Object.values(usuario);
console.log(valores); // ["Carlos", "carlos@email.com"]
3. Métodos para JSON
O objeto global JSON possui dois métodos principais que agem como tradutores. Eles convertem estruturas complexas em texto simples e vice-versa, permitindo o tráfego de dados pela rede de forma padronizada.
 
 
 
// Objeto base
const configuracao = { tema: "escuro", notificacoes: true };
 
// JSON.STRINGIFY: Transforma o Objeto em String (Texto)
// Ideal para enviar dados para um servidor
const dadosParaEnviar = JSON.stringify(configuracao);
console.log(typeof dadosParaEnviar); // Saída: "string"
 
// JSON.PARSE: Transforma a String (Texto) de volta em Objeto
// Ideal para receber dados de uma API e usar no sistema
const dadosRecebidosTexto = '{"versao": "1.2.0", "manutencao": false}';
const objetoPronto = JSON.parse(dadosRecebidosTexto);
console.log(objetoPronto.versao); // Saída: "1.2.0"
Dica de Arquitetura de Software
A escolha dos métodos afeta diretamente os requisitos não funcionais de performance. Métodos como push() e pop() operam no final do array e são muito rápidos. Já unshift() e shift() mexem no início do array, forçando a reorganização de todos os índices subsequentes, o que pode ser mais custoso em listas muito grandes.
