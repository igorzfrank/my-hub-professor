1. Listas (Arrays)
Em JavaScript, as listas são representadas pelos Arrays. Trata-se de um tipo de dado ordenado, que permite armazenar múltiplos valores em uma única variável. Os arrays em JavaScript são dinâmicos, podendo crescer ou diminuir de tamanho automaticamente, e podem armazenar diferentes tipos de dados simultaneamente.
 
 
 
// Criando uma lista de alunos
const alunos = ["Ana", "Carlos", "Beatriz"];
 
// Adicionando um novo elemento ao final da lista
alunos.push("Daniel");
 
// Acessando elementos pelo índice (começa em 0)
console.log(alunos[0]); // Saída: Ana
console.log(alunos.length); // Saída: 4
2. Dicionários (Objetos)
Dicionários são tipos de dados que armazenam informações no formato de chave-valor. Em JavaScript, a forma mais comum e idiomática de criar um dicionário é utilizando Objetos Literais. Cada informação é mapeada através de uma chave (nome da propriedade) única.
 
 
 
// Criando um dicionário de configuração de sistema
const configBancoDeDados = {
  host: "localhost",
  porta: 5432,
  usuario: "admin",
  ativo: true
};
 
// Acessando propriedades
console.log(configBancoDeDados.host); // Saída: localhost
 
// Adicionando uma nova chave-valor dinamicamente
configBancoDeDados.senha = "123456";
3. JSON (JavaScript Object Notation)
O JSON não é exatamente um tipo de dado de memória, mas sim um formato de texto utilizado para serializar e transmitir dados estruturados entre sistemas (como entre o Front-end e o Back-end em uma API). O JavaScript possui métodos nativos para converter Objetos em texto JSON e vice-versa.
 
 
 
const produto = { id: 101, nome: "Teclado", preco: 150.00 };
 
// 1. Convertendo um Objeto JavaScript em String JSON (Serialização)
const jsonTexto = JSON.stringify(produto);
console.log(jsonTexto);
// Saída em formato texto: {"id":101,"nome":"Teclado","preco":150}
 
// 2. Convertendo uma String JSON de volta para Objeto (Desserialização)
const respostaApi = '{"status": "sucesso", "codigo": 200}';
const objetoRecuperado = JSON.parse(respostaApi);
 
console.log(objetoRecuperado.status); // Saída: sucesso
Dica Prática de Desempenho
Ao definir tecnologias baseadas em requisitos não funcionais (como performance e uso de memória), lembre-se: Listas (Arrays) são excelentes para iteração e ordem sequencial, enquanto Dicionários (Objetos) são muito mais rápidos para buscas diretas através de uma chave específica.
