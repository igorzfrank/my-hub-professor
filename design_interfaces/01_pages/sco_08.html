1 📌 O QUE É UM SITEMAP?
🗺️ Definição simples
O Sitemap é um diagrama que organiza todas as páginas de um site em níveis hierárquicos, como uma árvore genealógica. No topo fica a página inicial (Home) e, conforme descemos, encontramos categorias, subcategorias e páginas cada vez mais específicas. É o primeiro passo antes de desenhar qualquer tela.
2 📝 O QUE VOCÊ DEVE ENTREGAR
🎯 Um fluxograma em estilo de árvore genealógica
Crie um diagrama que represente a estrutura de navegação de um e-commerce. O formato deve lembrar uma árvore genealógica: um elemento raiz no topo, que se ramifica em elementos filhos, netos e assim por diante. Use retângulos para as páginas e linhas para conectar os níveis hierárquicos. Seu fluxograma deve contemplar uma estrutura completa de loja virtual, com todos os tipos de página necessários para o funcionamento de um e-commerce.
💡 Lembre-se: Pense em lojas virtuais que você conhece. Quais páginas elas têm? Como as categorias se organizam? Use essas referências para montar sua árvore.
3 👀 EXEMPLO VISUAL — ÁRVORE GENEALÓGICA
🎯 Seu fluxograma deve seguir este estilo
Observe como as páginas se ramificam de cima para baixo, como uma árvore. A raiz é a Home e dela partem todas as outras. Cada nível abaixo representa páginas mais específicas.
4 📦 COMO FAZER E ENTREGAR
✅ Como fazer
🛠️ Use papel e caneta, PowerPoint, Figma ou qualquer ferramenta.

🌳 Monte no formato árvore genealógica: raiz no topo, ramificações para baixo.

🔲 Cada página é um retângulo com o nome dela dentro.

🔗 Conecte com linhas mostrando a hierarquia entre os níveis.
