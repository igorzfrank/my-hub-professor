1 🖼️ Edição de Imagens no Figma
🎯 Por que editar imagens no Figma?
O Figma não é apenas uma ferramenta de prototipagem — ele possui recursos poderosos de edição e manipulação de imagens que permitem ajustar fotos, aplicar máscaras, recortar e aplicar efeitos sem sair do ambiente de design. Isso agiliza o fluxo de trabalho e mantém tudo centralizado.
💡 Analogia simples
Editar imagens no Figma é como ter uma mini-suite de edição integrada ao seu estúdio de design. Você não precisa ir até outro software (Photoshop) para fazer ajustes rápidos — resolve ali mesmo, sem interromper o fluxo criativo.
⚙️Recursos de edição de imagens no Figma
✂️ Recorte (Crop)
Clique na imagem e arraste as alças para cortar. Use atalho Ctrl enquanto arrasta para recortar mantendo a proporção.
🎭 Máscaras (Mask)
Coloque a imagem dentro de uma forma. Use Ctrl+Alt+M para criar máscara. Ideal para fotos redondas ou com formas personalizadas.
🎨 Ajustes de Cor
No painel de propriedades: Exposure, Contrast, Saturation, Temperature, Tint, Highlights, Shadows.
✨ Efeitos (Effects)
Adicione sombras (Drop Shadow), desfoque (Blur), ruído e sobreposição de cores diretamente na imagem.
🔄 Redimensionamento
Arraste pelos cantos com Shift para manter proporção. Use o painel para definir valores exatos em pixels.
📥 Importação
Arraste imagens do computador, cole do clipboard (Ctrl+V) ou use plugins como Unsplash para fotos gratuitas.

2 🔷 Seleção de Elementos Visuais — Formas, Grafismos e Pictogramas
🎯 Elementos visuais no design de interfaces
Além de texto e imagens, as interfaces são compostas por formas geométricas, grafismos decorativos e pictogramas (ícones). Saber selecionar, combinar e editar esses elementos é essencial para criar interfaces visualmente ricas e profissionais.
🔷Formas Básicas no Figma
⬛
Retângulo (R)
Botões, cards, containers
⭕
Elipse (O)
Avatares, badges, ícones
🔺
Polígono
Setas, indicadores, gráficos
⭐
Estrela
Avaliações, favoritos, decoração
➖
Linha (L)
Separadores, divisórias, setas
✏️
Caneta (P)
Formas personalizadas, curvas
🎨Grafismos — Elementos Decorativos
✨ O que são?
Elementos visuais não informativos que complementam o design: bolinhas decorativas, linhas curvas, formas abstratas, gradientes e padrões de fundo.
🎯 Para que servem?
Criam personalidade visual, preenchem espaços vazios, guiam o olhar do usuário e reforçam a identidade da marca sem poluir a interface.
🔣Pictogramas — A Linguagem Visual Universal
🔣 O que são pictogramas?
Símbolos gráficos que representam objetos, ações ou ideias de forma simplificada. No design digital, são os ícones — a linguagem universal que transcende barreiras de idioma.
📂 Onde encontrar?
Plugins: Iconify, Phosphor Icons, Material Design Icons, Feather Icons. Bibliotecas: Font Awesome, Heroicons, Lucide.
💡 Boas práticas com pictogramas: Mantenha o mesmo estilo (outline ou filled), mesmo peso visual (espessura do traço) e mesmo tamanho de grid (ex: 24×24px) para todos os ícones do projeto. Consistência é tudo.
3 🧩 COMPONENTIZAÇÃO — O Poder dos Componentes no Figma
🎯 O que são Componentes?
Componentes são elementos reutilizáveis que funcionam como um "molde mestre". Quando você cria um componente (ex: um botão), todas as cópias (instâncias) se atualizam automaticamente se o principal for alterado. É o recurso mais poderoso do Figma para produtividade e consistência.
💡 Analogia simples
Componente principal é como o molde de um carimbo — cada vez que você "carimba" (copia), gera uma instância idêntica. Se você mudar o molde (cor, tamanho, texto), todos os carimbos já feitos se atualizam automaticamente.
📂Hierarquia dos Componentes
⭐ Componente Principal
O "molde mestre" (indicado por ♦ losango preenchido). Fica no arquivo e define o padrão para todas as cópias.
📋 Instância
Cópia do componente principal (indicado por ♦ losango vazado). Pode ter pequenas variações (cor, texto).
🎨 Variantes
Versões diferentes do mesmo componente agrupadas (ex: botão primary, secondary, disabled).
📚 Biblioteca
Coleção de componentes compartilhada entre diferentes arquivos da equipe.
⚙️Como criar componentes e variantes
Passo 1 — Criar Componente
Selecione o elemento (botão, card, input) e pressione Ctrl+Alt+K ou clique em "Create Component" na barra superior.
Passo 2 — Criar Variantes
Crie variações do componente (ex: cor diferente, tamanho diferente). Selecione todos e clique em "Combine as Variants" no painel direito.
Passo 3 — Usar Instâncias
Arraste o componente do painel Assets para o canvas. No painel direito, alterne entre variantes e edite propriedades (texto, cor).
💡 Exemplo prático: Crie um componente "Botão" com cor azul e texto "Clique aqui". Use 50 instâncias em 20 telas. Depois, mude a cor do principal para verde. Todas as 50 instâncias se atualizam instantaneamente.

5 ✅ Boas Práticas e Fluxo de Trabalho
✅ Boas Práticas
✅ Nomeie componentes com clareza (ex: Button/Primary/Default)
✅ Use Auto Layout em componentes para responsividade
✅ Crie variantes em vez de componentes separados
✅ Use ícones como componentes (biblioteca de ícones)
✅ Organize componentes em páginas separadas no Figma
✅ Documente propriedades editáveis de cada componente
❌ Erros Comuns
❌ Criar componentes sem nome (Rectangle 237, Group 42)
❌ Copiar e colar em vez de usar instâncias
❌ Não usar Auto Layout (botões que não ajustam ao texto)
❌ Misturar estilos de ícones (outline + filled)
❌ Componentes com tamanhos inconsistentes
❌ Esquecer de criar variantes (muitos componentes soltos)
