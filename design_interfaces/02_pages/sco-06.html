1 🎯 CONTEXTO — Por que replicar interfaces?
🧠 Aprender fazendo
Replicar interfaces existentes é uma das melhores formas de aprender design. Ao reproduzir uma tela no Figma, você desenvolve o olhar para detalhes como espaçamentos, alinhamentos, hierarquia visual e proporções. É como um músico que aprende tocando músicas de outros artistas antes de compor as suas. Nesta atividade, você vai replicar duas interfaces fornecidas como referência.
💡 Habilidades desenvolvidas
Nesta atividade você praticará: observação detalhada (identificar cada elemento da interface), uso de ferramentas do Figma (formas, textos, cores, alinhamento), organização de camadas e atenção aos espaçamentos e proporções corretas.

3 ⚙️ COMO EXECUTAR A ATIVIDADE
✅ Passo a passo
🔍 1. Observe a imagem de referência com atenção antes de começar.
📐 2. Identifique os elementos principais: textos, botões, imagens, ícones.
🎨 3. Recrie cada elemento usando as ferramentas do Figma.
📏 4. Ajuste espaçamentos, cores e tamanhos para ficar fiel.
🔤 5. Confira fontes, pesos e estilos tipográficos.
📂 6. Organize as camadas com nomes descritivos.
⚠️ Pontos de atenção
❌ Não deixe elementos desalinhados — use as guias e grades do Figma.
❌ Não use cores aproximadas — use o conta-gotas ou copie o código hexadecimal.
❌ Não ignore os espaçamentos — padding e margens fazem diferença.
❌ Não esqueça das camadas — nomeie cada frame e grupo.
❌ Não entregue com fontes diferentes — encontre a fonte mais próxima possível.
💡 Dica: Use o atalho Ctrl + G (ou Cmd + G no Mac) para agrupar elementos relacionados. Use Alt + arrastar para duplicar elementos rapidamente. E use as guias (arraste a partir da régua) para garantir alinhamento perfeito.
4 📦 COMO FAZER E ENTREGAR
🔗 Como gerar o link do Figma
1. No Figma, abra o arquivo com suas interfaces.
2. Clique no botão "Share" (canto superior direito).
3. Em "Anyone with the link", selecione "can view".
4. Clique em "Copy link" para copiar o link.
5. Cole o link no documento PDF de entrega.
📄 Forma de Entrega
🧑‍💻 Atividade individual — Cada aluno faz o seu.

🔗 Link do Figma — Gere o link de visualização do seu arquivo.

📄 Formato PDF — Crie um PDF contendo o link do Figma.

📤 Envio online — Envie o PDF pelo ambiente virtual da disciplina.
📊 Critérios de Avaliação
✅ Fidelidade visual — A réplica está fiel à imagem de referência?
✅ Cores e tipografia — As cores e fontes foram reproduzidas corretamente?
✅ Espaçamentos e alinhamentos — Os elementos estão bem posicionados?
✅ Organização das camadas — Os frames e grupos estão nomeados?
✅ Link funcional — O link do Figma está acessível e com permissão de visualização?
💡 Dica final: Antes de enviar, teste o link do Figma em uma janela anônima do navegador. Se você conseguir visualizar o arquivo sem estar logado, o professor também conseguirá. Não se esqueça de verificar a permissão! 🚀
