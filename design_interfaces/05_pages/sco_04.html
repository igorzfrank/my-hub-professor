🔧 Configuração Inicial
Passo 1: O Layout Grid é essencial para manter tudo alinhado.

Antes de ativar o grid, você precisa de um Frame.

  1. Aperte a tecla F ou clique no ícone de # na barra superior.

  2. No painel direito, escolha um tamanho (ex: Desktop > MacBook Air).

Passo 2: Ativar o Layout Grid

Com o Frame selecionado:

  1. Vá até o painel lateral direito.
  2. Procure a seção escrita Layout Guide.
  3. Clique no ícone de "+" (mais).



Passo 3: Configurar para "Colunas" (O padrão de Landing Pages)
Por padrão, o Figma cria uma grade de quadradinhos vermelhos. Vamos mudar para colunas:
  1. Clique no ícone de 9 quadradinhos (ou no nome "Grid") que apareceu dentro da seção Layout Grid.
  2. No menu suspenso que abrir, onde diz "Grid", mude para Columns.

💡 Dica de Ouro: Margens e Respiros
Se você optar pelo tipo Stretch (que estica), defina margens laterais entre 100 a 150. Isso garante que o conteúdo não encoste nas bordas do monitor, criando um "respiro" visual elegante e profissional.
